Microsoft Expression Web 2 : das offizielle Trainingsbuch
Microsofts Webeditor "Expression Web 2" ist wohl der meistgenutzte Bestandteil der Produktfamilie "Microsoft Expression Studio 2" zur Erstellung und Verwaltung auch größerer Websites. Der Nachfolger des früheren Editors "Microsoft FrontPage" wird hier von Chris Leeds vorgestellt, der eng mit Microsoft verbunden ist und auch die Vorauflage dieses Buchs unter dem Titel "Microsoft Expression Web Step by Step" (hier nicht besprochen) geschrieben hat. Er betont: "Das alles überragende Ziel von Expression Web 2 besteht darin, Code zu erzeugen, der den W3C-Richtlinien entspricht und der einen barrierefreien Zugang zu den Inhalten ermöglicht". Dank großzügigem Layout des Buchs mit vielen (einfarbigen) Bildschirmabbildungen kann man die einzelnen Trainingsabschnitte gut verfolgen. Außerdem hilft die beigefügte CD-ROM, die alle Übungsdateien enthält, aber auch den Inhalt des gut strukturierten Bandes als E-Book. - Eine notwendige Warnung: Für das Training und dann natürlich auch für den Editor selbst muss Windows Vista oder Windows XP mit Servicepack 2 oder 3 verfügbar sein.
