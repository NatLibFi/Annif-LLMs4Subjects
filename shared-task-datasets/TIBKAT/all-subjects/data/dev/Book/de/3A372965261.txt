ADO.NET : einfache Datenbankprogrammierung im .NET-Framework ; [Start ohne Vorwissen ; alle Beispiele aus dem Buch inklusive einer kompletten Beispielapplikation (Pizza-Service)]
ADO.NET ist die Datenzugriffskomponente in dem von Microsoft neu eingeführten .NET Framework. ADO steht für Active Data Objects. ADO.NET unterscheidet sich beträchtlich von früheren ADO-Versionen. ADO.NET spielt auch eine wichtige Rolle für Web-Services. Das Buch setzt beim Leser keine besonderen Vorkenntnisse voraus. Es beginnt mit einer ausführlichen Theorie der objektiorientierten Datenmodellierung und zeigt auf, welche Auswirkungen eine objektorientierte Datenzugriffs-API auf den Entwurf von Datenbanken hat. Hiermit wird auch deutlich, wie sich ADO.NET von früheren ADO-Versionen unterscheidet. Es folgt eine Einführung in ADO.NET-Klassenbibliotheken, die als nützliche Werkzeuge mit kleinen Code-Beispielen erläutert werden. Am Schluss steht als große Anwendungsapplikation die Programmierung von PizzaBlitz, einem Pizza-Liefer-Service. Im Ansatz grundverschieden von R. Riordan (ID 34/03). Das vorliegende Buch ist teilweise frech formuliert und fordert den Leser immer wieder zum Mitdenken heraus, z.B. mit der Frage "Müssen Datenbanksysteme wirklich sein?". (2)
