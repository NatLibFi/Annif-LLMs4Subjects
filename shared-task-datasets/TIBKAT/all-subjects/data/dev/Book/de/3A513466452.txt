Java EE 5 : Einstieg für Anspruchsvolle ; auf CD: JBoss, Apache Tomcat und Java EE
Nachdem die Java 2 Standard Edition bereits seit geraumer Zeit als Version 5 vorliegt, wurde nun auch deren Enterprise Edition, die nicht nur eine Programmiersprache, sondern ein ganzes Paket von Applikationen und Schnittstellen für webbasierte Unternehmensanwendungen darstellt, auf diesen Stand gebracht. Dazu hat Autor Stark seinen Reihenvorgänger zu J2EE (ID 6/05) intensiv überarbeitet und um 80 Seiten erweitert. Behandelt neben den Grundlagen der Programmiersprache weitere aktuelle Java-Technologien wie EJB, JSP, JSF und Struts, geht auf Tag-Bibliotheken, XML, XSL, XPath, Web Services und weitere Side Solutions ein. Richtet sich vor allem an professionelle Softwarentwickler mit erheblichen Vorkenntnissen, kann aber angesichts der starken Verbreitung von Java EE 5 inzwischen schon für mittlere EDV-Bestände empfohlen werden. Der Titel der bei Münchner Benutzern ausgesprochen gut ankommenden Reihe, ist der 1. zur aktuellen Version überhaupt. Die Beilage-CD-ROM enthält einen Lerntest, Beispiele aus dem Buch und einige sinnvolle Open-Source-Programme. Der Vorgänger kann verbraucht werden. (2)
