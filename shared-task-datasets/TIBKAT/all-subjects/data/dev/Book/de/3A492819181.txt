Data Becker's Buch zu SUSE LINUX : [die Windows-Alternative! ; so gelingt der Umstieg von Windows im Handumdrehen ; das LINUX-Paket mit allem, was Sie für unbeschwertes Arbeiten wirklich brauchen! ; Praxishandbuch für den schnellen Einstieg mit SUSE 9.3 Special Edition auf DVD]
Linux wird immer stärker auch für Privatanwender zu einer Alternative zum weitverbreiteten Betriebssystem Windows von Microsoft. Der Autor legt ein Praxisbuch vor, das Windows-Anwendern einen leichten Übergang auf Linux ermöglichen soll , oder auch - zum leichteren Umgewöhnen - einen Parallelbetrieb von Windows und Linux auf einem Rechner. Beigegeben ist auf DVD eine Spezial Edition mit den wichtigsten Programmen der Linux Distribution Suse 9.3 (mit kostengünstiger Update-Möglichkeit auf die Vollversion). Als Live-DVD eingesetzt erübrigt sich eine aufwendige Installation; als grafische Benutzeroberfläche wird KDE benutzt. Mit zahlreichen kurzen Workshops (sehr oft auch Schritt-für-Schritt-Anweisungen) wird der Leser in das Arbeiten mit Suse Linux eingeführt. Zusätzlich gibt es zahlreiche Vergleiche mit Windows (z.B. Auflistungen von Linux-Analoga zu häufig eingesetzten Windowsprogrammen). Auch ganz aktuelle Themen wie das Telefonieren über das Internet (VoIP) werden gut berücksichtigt. Zur Weiterführung und Vertiefung sei u.a. auf G. Wielage (BA 1/05) hingewiesen. (1)
