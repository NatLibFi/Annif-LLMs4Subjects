Microsoft Windows XP Professional - das Handbuch
Einführung in die Features von Windows XP, wobei der Schwerpunkt auf der Installation von Geräten und Programmen liegt, der Nutzung digitaler Medien, dem Umgang mit Netzwerken, dem Internet sowie Performance und Sicherheit; mit vielen Screenshots. (Klaus Barckow)
Der Autor gilt als Spezialist für die Betriebssystemfamilie Windows von Microsoft und ist bekannt durch Bücher zu früheren Versionen (zuletzt BA 8/01 - Windows 2000 professional; BA 12/00 - Windows ME). Hier liegt eine didaktisch gut aufgebaute, umfassende Beschreibung der aktuellen Version vor, die keine Wünsche offen lässt. Viele Bildschirmabbildungen und tabellarische Übersichten. Die vielen Neuerungen von Windows XP sind eingangs zusammengefasst; von hieraus Hinweise, wo welche Features ausführlich behandelt werden. Das Buch eignet sich somit auch für Umsteiger von älteren Versionen. Ein tiefgegliedertes Inhaltsverzeichnis und ein umfangreiches Stichwortverzeichnis ermöglichen einen punktuellen Einstieg und helfen bei plötzlichen Problemen weiter. Auf CD-ROM u.a. die Windows Power Tools als zusätzliche Hilfe bei der Konfiguration. Schwerpunkte liegen bei der Installation von Geräten und Programmen, bei der Nutzung digitaler Medien, beim Umgang mit Netzwerken, beim Arbeiten im Internet und bei den Aspekten Performance und Sicherheit. Nach J. Schieb (BA 5/02), parallel zu A. Maslo (BA 1/02). (2) (Klaus Barckow)
