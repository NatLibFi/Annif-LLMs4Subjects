Windows-Testumgebung : für Ausbildung und Beruf ; [Windows Server 2008, Exchange Server 2007, Active Directory, ISA Server 2006, Clustering]
Der Autor ist u.a. bekannt durch Bücher über den Windows Server 2008 (ID 35/08) und den Exchange Server 2007 (ID 26/07 - hier schon kurze Ausführungen über Testumgebungen). In einem Handbuch über Microsoft-Netzwerke (ID 49/06) hat er sich intensiv mit Installation, Wartung und Verwaltung eines Active Directory sowie mit ISA Servern (Firewall) beschäftigt. Er ist somit Spezialist für Netzwerke. Im vorliegenden Buch weist er darauf hin, wie wichtig Testumgebungen auch für Netzwerke sind. Mittels dieser Testumgebungen können Softwareprogramme und Installationsprozesse getestet, aber auch Abläufe in Netzwerken simuliert und Fehlerquellen aufgespürt werden. Der Autor demonstriert die Einrichtung von Testumgebungen und die Auswertung eventueller Fehlermeldungen, sowohl für Stand-Alone-Rechner als auch für Cluster. Von großer Bedeutung für ihn ist die sorgfältige Trennung Produktionsumgebung und Testumgebung, um Schäden für die Produktionsumgebung zu vermeiden. Er setzt hierzu Virtualisierungssoftware ein, wobei er VMware Workstation (als Testversion auf Begleit-CD) bevorzugt. - Auch für Einsteiger. (2)
