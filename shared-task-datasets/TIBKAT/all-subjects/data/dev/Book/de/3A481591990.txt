Delphi 2005 : [Delphi programmieren mit dem .NET Framework]
Die .NET-Technologie ist eine neue Softwaretechnologie von Microsoft. Bereitgestellt wird u.a. eine Plattform, auf der Programme - in verschiedenen Sprachen geschrieben - problemlos zusammenarbeiten können sowie eine sprachübergreifende Klassenbibliothek (FCL). Delphi 2005 ist eine neue Version von Delphi, die die Programmierung mit dem .NET-Framework ermöglicht. Der Autor gilt als Spezialist für Delphi (zuletzt ID 7/04 - Delphi 6; ID 8/02 - Kylix, Delphi für Linux). Hier stellt er das Programmieren mit Delphi 2005 vor. Angesprochen werden Leser mit allgemeinen Vorkenntnissen in der Programmierung, z.B. in Java, Visual Basic, C, C++ oder C@Einsetzbar für alle Editionen von Delphi 2005; allerdings werden dbWeb-Steuerelemente und das ECO-Framework nicht behandelt. Gut herausgearbeitet werden die Unterschiede zu früheren Versionen von Delphi. Schwerpunkte liegen bei einem Vergleich der .NET-Framework-Klassenbibliothek FCL mit der Delphi-eigenen Klassenbibliothek VCL, bei Datenbankanwendungen und bei der Entwicklung neuer Komponenten. Viele gut kommentierte Beispiele. 28 Seiten Stichwortverzeichnis. (2)
