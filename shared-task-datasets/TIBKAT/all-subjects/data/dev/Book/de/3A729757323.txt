Jetzt lerne ich HTML : Start ohne Vorwissen
In unregelmäßigen Abständen erscheint in der "jetzt lerne ich"-Reihe eine Einführung in HTML. An der zuletzt angezeigten Ausgabe (BA 5/09), für die ebenfalls Prevezanos verantwortlich war, gab es berechtigte Kritik bezüglich einer unsauberen Darstellung beim Zusammenspiel von HTML und CSS. Erfreulicherweise liegt jetzt eine gründlich überarbeitete Ausgabe vor. Der Inhalt präsentiert sich deutlich klarer strukturiert und die Beschreibung der Funktionalitäten hat an Präzision gewonnen. Einsteiger lernen die neuesten Standards von HTML 5, soweit sie abgesichert sind. Bei Bedarf wird auf HTML 4 zurückgegriffen. Das CSS-Kapitel hat an Umfang zugelegt und die Rollenverteilung zwischen HTML und CSS wird gut herausgearbeitet. Leider fehlt ein Hinweis darauf, ob und in welchem Umfang Funktionalitäten von CSS 3 berücksichtigt werden. Am Ende eines jeden Kapitels befinden sich Übungen, deren Lösungen im umfangreichen Anhang zu finden sind. Alle Schwarz-Weiß-Abbildungen sind von einwandfreier Qualität. Neben Th. Kobert (ID-A 46/12) und vor J. Ducket (ID-B 4/13). (1) (LK/HB: Isigkeit)
