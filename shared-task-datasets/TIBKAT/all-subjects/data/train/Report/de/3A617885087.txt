Netzwerke für Dummies : [Netzwerker netzwerken in ihrem eigenen Netzwerk ; auf einen Blick: Implementierung, Administration, Troubleshooting und Sicherheit ; Netzwerke über LAN und WLAN einrichten ; Netzwerke vor Hackern schützen ; Netzwerk-Betriebssysteme - Windows 7, Windows Server 2008 R2, Mac OS X und Co.]
Nun bereits in der 6. Auflage vermittelt der Autor wieder auf unterhaltsame und verständliche Weise alles Wissenswerte zum Thema Netzwerke. Ein Blick auf und in das Buch zeigt, dass alle Ingredienzien der bekannten eingeführten Reihe wieder vertreten sind. Auch der überwiegende Teil des Textes stimmt mit der Vorauflage (BA 10/08) überein. Die Veränderungen und Aktualisierungen beziehen sich auf die Berücksichtigung des neuen Betriebssystems Windows 7 und den Windows Server 2008 Release 2. Ausserdem hat Lowe ein neues Kapitel zum Umgang mit Mobilgeräten eingefügt. Wobei er sich auf Windows Mobil- und BlackBerry-Geräte konzentriert. Auch der obligate Top-Ten-Teil erhielt eine Auffrischung. Zur Bestandsaktualisierung weiterhin wärmstens empfohlen. Der Vorgänger kann auslaufen, ältere Werke sollten jedoch ersetzt werden. (2)
