Linux Mint 19 : Praxiswissen für Ein- und Umsteiger
Linux Mint 19 (Tara) ist die aktuelle Version einer Linux-Distribution, die auf Linux Ubuntu beruht. Der Autor ist u.a. bekannt durch ein Buch über die Vorgängerversion Linux Mint 18. Das Buch setzt bei seinen Lesern keine besonderen Vorkenntnisse voraus. Angesprochen werden insbesondere Umsteiger von Windows, denen mit Linux Mint ein Betriebssystem angeboten wird, das in seiner Leistungsfähigkeit gut mit Windows vergleichbar ist, das aber kaum Hacker-Angriffen ausgesetzt ist. Der Autor beginnt mit einer kurzen Beschreibung von Linux Mint und der Installation, wobei auch auf die Möglichkeit eingegangen wird, Linux Mint und Windows parallel auf einem Rechner zu betreiben. Weitere Schwerpunkte liegen dann bei der Einrichtung von Linux Mint, dem Bürosoftwarepaket LibreOffice, Linux Mint als Mehrbenutzersystem, Linux Mint in der Cloud und der Datensicherheit. Viele Abbildungen; didaktisch gut aufbereitet: Hinweise, Regeln und Tipps sind grafisch hervorgehoben. Eine Beschaffung zusätzlich zur o.a. Ausgabe Linux Lint 18 wird empfohlen wegen vieler Neuerungen, z.B. in der Musikverwaltung
