Microsoft Exchange Server 2019 : das Handbuch für Administratoren
Microsoft Exchange Server ist eine weitverbreitete Groupware für die Zusammenarbeit in einer Gruppe, insbesondere den Austausch von Nachrichten. Das Buch hier beschreibt die seit Anfang dieses Jahres verfügbare Version 2019 und ist somit ein guter Ersatz für die Handbücher von Th. Joos, die früheren Versionen (Version 2016) beschreiben. Angesprochen werden Leser mit Vorkenntnissen; aufgrund der gut erklärten zahlreichen Beispiele ist aber auch ein Zugriff für Einsteiger möglich. Der Quellcode aller Beispieldateien findet sich zum Download im Netz. Der Autor beginnt kurz mit den wesentlichen Neuerungen der aktuellen Version (Hochverfügbarkeit, Unterstützung von Windows Server 2019 Core, neue Suchmaschine, neue Metacache-Datenbank, Wegfall der Unified Messaging-Funktionen) und setzt dann Schwerpunkte bei Konfiguration. Administration, Betrieb und Wartung des neuen Exchange Servers. Sehr hilfreich sind zum Schluss auch Anmerkungen über Best Practices und Beispiele "ungünstiger" Exchange-Implementierungen
