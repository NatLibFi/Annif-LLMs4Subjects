Let's code Python
Der Autor ist u.a. bekannt durch ein Buch über die Programmierung von Apps, das keinerlei Vorkenntnisse voraussetzt und sich sehr gut im Schulunterricht einsetzen lässt. Das Buch hier ist in erster Linie auch für den Schulunterricht gedacht, setzt ebenfalls keine Vorkenntnisse voraus und führt gestützt auf die Sprache Python mit ihrer einfachen Syntax in die Grundprinzipien des Programmierens ein. Eingesetzt wird die Entwicklungsumgebung TigerJython (beigefügt auf DVD) mit ihren vielen Bibliotheken; der Einstieg erfolgt somit über einen Computer mit richtiger Tastatur, auf dem sogar Grafik und akustische Möglichkeiten verfügbar sind. Für das Arbeiten auf mobilen Geräten wird auf der Webseite zum Buch auf das kostenfreie WebTigerJython hingewiesen. Der Autor beginnt mit den Grundbegriffen der Programmierung (Befehle, Module, Schleifen, Objekte, ...) und kommt dann zur Umsetzung dieser Begriffe in Spielen (gamegrid, Breakball, Space Attack, TicTacToe, ...). Zusammen mit dem o.a. Buch eine gute Möglichkeit, im Unterricht mit den Grundlagen der digitalen Welt vertraut zu machen
