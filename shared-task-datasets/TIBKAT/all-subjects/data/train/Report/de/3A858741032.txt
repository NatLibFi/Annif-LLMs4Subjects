PHP 7 und MySQL : das umfassende Handbuch
Das bekannte Autorenduo hat sein umfangreiches Handbuch zur Gestaltung von dynamischen Websites mit PHP und MySQL aktualisiert und überarbeitet. Am Anfang stehen die notwendigen Installationsschritte, gefolgt von einer ausführlichen Darstellung der Sprachgrundlagen. Zahlreiche Codebeispiele ergänzen die Ausführungen. Praxisgerechte Anwendungsbeispiele folgen im 3. Schritt. Die eingefügten Schwarz-Weiss-Abbildungen stammen zwar überwiegend von Windows-Plattformen, gleichwohl kann der Inhalt auch für Linux- und Mac-Systeme genutzt werden. Diese Ausgabe behandelt die gründlich modernisierte Hauptversion 7, die neben einer 64-Bit-Unterstützung auch einige neue Features mitbringt. Hierfür wurden Kapitel erweitert, u.a. zu JavaScript (Berücksichtigung der Websockets) und zu MySQL (ergänzt um fortgeschrittene Features wie z.B. BLOBs). Gänzlich neu ist das Kapitel zur Verwendung des Paket- und Abhängigkeitsmanagers Composer. Entfallen sind Aussagen zur PHP-Erweiterungsbibliothek PEAR. Die Version 7 ist kompatibel zur Version 5.x, sodass der Vorgänger noch weiter nutzbar ist
