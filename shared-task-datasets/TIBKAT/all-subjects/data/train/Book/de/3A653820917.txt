Python von Kopf bis Fuß : [ein Buch zum Mitmachen und Verstehen]
Python ist eine dynamische Programmiersprache, die es u.a. ermöglicht, Programme in anderen Sprachen als Module einzufügen. Eine umfassende Darstellung für Einsteiger mit Vorkenntnissen ist M. Pilgrim (BA 12/10). Der Autor ist u.a. bekannt durch ein Buch in der Reihe "von Kopf bis Fuss" über das Programmieren (BA 8/10), das sich in seinen Beispielen auf Python stützt. In der reihentypischen Vorgehensweise führt der Autor auch Einsteiger an praxisnahen Beispielen direkt in das Arbeiten mit Python ein. Sprachelemente und Syntax werden bedarfsweise erläutert. Gleich zu Beginn kommt der Leser mit Listen in Kontakt, einem wichtigen Arbeitsmittel von Python. Auch wird auf ein arbeitsteiliges Vorgehen durch das Einstellen von Modulen in der Python-Software-Distributionssite hingewiesen. Weitere Schwerpunkte liegen beim Debugging und beim Datenbankzugriff mittels SQLite. Gut berücksichtigt wird die Entwicklung von Apps für Android-Smartphones. Einen Hinweis auf Themen, wo eine spätere Vertiefung sinnvoll ist, bietet die Auflistung der 10 wichtigsten Dinge, die der Autor nicht behandelt hat. (2)
