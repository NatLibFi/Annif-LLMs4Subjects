Microsoft .NET 2.0 Crashkurs : [Schnelleinstieg in neue Technologien und Tools ; DVD mit Testversion von Visual Studio]
Microsofts komplexe Softwareentwicklungsinfrastruktur .NET bietet eine moderne und flexible Umgebung zur Entwicklung von Software aller Art. Hiermit liegt eine Einführung in das .NET Framework 2.0, das .NET Framework Development Kit und die Entwicklungsumgebung Visual Studio 2005 vor, mit der die Ausgabe aus dem Vorjahr (ID 33/05) aktualisiert wird, die sich noch auf die Betaversion bezog. Als Konsequenz aus dem Crashkurs-Ansatz des Buches ergibt sich eine komprimierte und in der Themenauswahl selektive Darstellung. Ziel des Autors ist es, einen Schnelleinstieg in das neue Produkt zu liefern, daher konzentriert sich der Text auf die zentralen Konzepte und die Einsatzmöglichkeiten anhand ausgewählter Beispiele. Angesprochen werden Entwickler, die als Umsteiger oder Erstanwender die neue Technologie nutzen möchten. Die Neuerungen der aktuellen Version werden im Text hervorgehoben und kurz beschrieben. Auf der beigefügten DVD-ROM befindet sich eine 180-Tage-Version von Visual Studio 2005 (nur nutzbar mit dem Product Key, unbedingt sichern).(3)
