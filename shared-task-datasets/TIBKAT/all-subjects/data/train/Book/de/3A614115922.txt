Mac OS X Snow Leopard : das missing manual ; das fehlende Handbuch zu ihrer Software
Der Autor ist bekannt durch seine Bücher "missing manuals", die heute oft fehlende gedruckte Betriebsanleitungen ersetzen sollen (z.B. BA 6/08 - iPhone; ID 40/04 - Mac OS X 10.3 Panther). Hier legt er ein entsprechendes Buch über die aktuelle Version des Mac-Betriebssystems, OS X 10.6 vor. Wie gewohnt flott und gut nachvollziehbar geschrieben; umfangreicher noch als bisher besprochene Standardwerke wie D. Mandl (BA 10/04), Sc. Kelby (BA 2/10) oder das Videotraining "Mac OS A 10.6 - Snow Leopard" (BA 3/10). Angesprochen werden Einsteiger, aber auch Fortgeschrittene und Mac-Fans. Grafisch hervorgehobene Hinweise auf unterschiedliche Schwierigkeitsgrade und Interessen ("Frequently asked questions", "Im höchsten Gang", "Sprechstunde für Power-User", "Zuckerstückchen"). Schwerpunkte liegen beim Finder und beim neuen QuickTime-Player. Gut berücksichtigt werden aber auch kostenfreie Mac-Programme, die Fehlersuche, das Arbeiten im Internet und der Einsatz von Windows auf dem Mac. Im Anhang hilfreiche Hinweise zur Installation, eine Liste der Mac OS X-Tastaturkürzel und ein Wörterbuch Windows - Mac. (2)
