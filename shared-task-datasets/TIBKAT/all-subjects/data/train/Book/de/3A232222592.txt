Windows NT 4 - das Kompendium : Workstation und Server in einem Band - gründlich aktualisiert und mit den heißesten Profithemen: die Windows NT Registry, MS Backoffice und NT Server, Aufbau eines Internet-Servers
Der Schwerpunkt des Buches liegt auf dem Windows NT-Server als zentraler Komponente eines Netzwerkes mit Client-Server-Applikationen und Internet-/Intranet- Schnittstellen. Gezeigt werden die Installation eines NT-Servers und die Anbindung der Clients. Domänenkonzepte und die Anbindung an Netware- und TCP-IP-Umgebungen sind auch Schwerpunkte. Weitere Themen werden unter folgenden Überschriften behandelt: Windows NT - Grundlagen, Planung und Konzeption des Windows NT- Netzwerks, Installation und Inbetriebnahme, Konfiguration und Feinabstimmung, Administration des Windows NT Servers, Windows NT Basic Services, Windows NT- Networking, Internet-Intranet-Working mit Windows NT, Client-Computing bzw Server- Computing mit Windows NT-Server, Desktop-Applikationen unter Windows NT, Fragen und Antworten zu Windows NT, Service Packs, Ausblick - Windows NT 5-0 und weitere Entwicklungen, Befehle von Windows NT.
