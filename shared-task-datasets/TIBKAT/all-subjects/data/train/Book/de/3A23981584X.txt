HTML 4.0 für Profis
HTML (HyperText Markup Language) ist eine Seitenbeschreibungssprache, die plattformunabhängig die Gestaltung von Web-Seiten ermöglicht. Über Hyperlinks bietet HTML auch die Möglichkeit, Dokumente miteinander zu vernetzen. Der Autor ist bekannt durch eine Einführung in die Version HTML 3.2 (BA 5/97). Hier stellt er die aktuelle Version HTML 4.0 vor. Das Buch ist als methodischer Einstieg gedacht, kann aufgrund der vielfältigen Zugriffsmöglichkeiten aber auch als Referenz verwendet werden. Mit vielen Listings wird die Syntax von HTML erläutert. Formatierung, Vernetzung, Tabellen und Listen, das Einfügen von Objekten, Formulare und Frames werden in eigenen Kapiteln behandelt. Hilfreich sind Hinweise auf Browser-spezifische Besonderheiten und auf Sprachelemente früherer Versionen, die zwar noch unterstützt, aber nicht mehr empfohlen werden. Ausführlich beschäftigt sich der Autor mit Style Sheets (Vorlagen für die Gestaltung von Web-Seiten). Klare, nüchterne, didaktisch gut strukturierte Darstellung. Vor L. Lemay (ID 16/98).
