Datenbanken entwickeln mit SQL Server 2008 Express Edition : [mit SQL Server 2008 Express Edition auf DVD ; alles was Einsteiger benötigen: viele Übungen und Insider-Tipps ; Hintergrundwissen und Konzepte ; Software zum sofortigen Loslegen]
SQL Server ist ein relationales Datenbankmanagementsystem von Microsoft. Die aktuelle Version 2008 wird - auch für Einsteiger und Umsteiger von älteren Versionen - sehr gut von K. Konopasek (BA 8/08) dargestellt. Das vorliegende Buch wendet sich ebenfalls an Einsteiger ohne größere Vorkenntnisse. Es stützt sich auf die kostenfreie SQL Server 2008 Express Edition (auf Begleit-DVD beigefügt) und vermittelt Datenbankgrundlagen und Programmierhinweise. Der Leser wird so in die Lage versetzt, Datenbankprojekte zu realisieren, die Access-basierten Datenbanken in der Funktionalität gleichkommen. Die neuen Features der aktuellen Version werden dabei gebührend herausgestellt. Didaktisch gute Aufbereitung. Jedes Kapitel beginnt mit Lernzielen. Dann folgen ein Schritt-für-Schritt-Vorgehen zur Realisierung des Lernzieles, Übungen und eine Zusammenfassung. Das Buch benutzt eine durchgehende Beispieldatenbank, die immer weiter entwickelt wird (Mediendatenbank zur Verwaltung von Büchern und CDs/DVDs). Viele Tipps, Hintergrundinformationen und Links. Im Anhang gute Hinweise auf SQL-Server-Foren und -Newsgroups. (2)
