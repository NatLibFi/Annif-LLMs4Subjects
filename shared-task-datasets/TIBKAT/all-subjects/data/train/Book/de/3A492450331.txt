Windows XP Professional : Profiwissen, Konfiguration, Netzwerke ; [Service Pack 2]
Bereits in 3. Auflage erscheint dieser stattliche Band aus der eingeführten Reihe zum Betriebssystem Windows XP Professional (Vorauflagen im ID nicht angezeigt). Monadjemi, ausgewiesener Kenner der Materie, präsentierte erst vor kurzem eine neue Auflage zur Version Windows XP Home (BA 3/05, gleiche Reihe) und Tierling gehört seit Jahren zu den Spezialisten für die Themen Netzwerke und Betriebssysteme. Anlass für eine Neuausgabe ist das Service-Pack 2, für das es, neben Ergänzungen im lfd. Text, auch ein eigenes Kapitel gibt. Zudem erhielt die Windows Instrumentation Management-Schnittstelle (WMI) nunmehr ein eigenes Kapitel. Das breit angelegte Grundlagenwissen blieb jedoch unverändert erhalten und demzufolge muss die Vorauflage, soweit vorhanden, nur dort ersetzt werden, wo differenzierte Darstellungen zum Thema Service-Pack 2, wie die Titel von Tobias Weltner (BA 12/04) oder Andreas Maslo (BA 4/05), fehlen. Als Erstanschaffung für den fortgeschrittenen XP-Anwender, der sich tiefer in das Thema einarbeiten möchte, gut geeignet.(2)
