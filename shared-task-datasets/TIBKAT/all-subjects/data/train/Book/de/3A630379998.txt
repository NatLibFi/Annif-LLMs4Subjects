Das Einsteigerseminar Projektmanagement mit Microsoft Project 2010 : [Termine, Kosten & Ressourcen im Griff ; lernen, üben, anwenden]
Einführung für Einsteiger in die wichtigsten Arbeitsverfahren der Projektplanungssoftware Microsoft Project 2010. Die Schwerpunkte liegen bei den Themen Projektstrukturplan, Projektablaufplan, Ressourcenplan und Kostenplan. Viele Schritt-für-Schritt-Anweisungen. Großes Fallbeispiel. (Klaus Barckow)
Microsoft Project 2010 ist die aktuelle Version einer auf Netzplantechnik basierenden Software zum Planen, Steuern und Überwachen von Projekten und ist Bestandteil der Office-Familie 2010. Die Autorin ist bekannt durch Einführungen in frühere Versionen (vgl. BA 7/08 - Microsoft Project 2007, kann noch im Bestand bleiben). - In diesem Buch werden Einsteiger ohne besondere Vorkenntnisse angesprochen. Das Buch beschränkt sich auf Grundlagen. Leser mit umfangreicheren Kenntnissen seien auf "Microsoft Project Server 2010" (ID 48/10) hingewiesen, wo auch sehr ausführlich auf die zentrale Verwaltung mehrerer Projekte und ihrer Ressourcen eingegangen wird. Schwerpunkte liegen bei den Themen Projektstrukturplan, Projektablaufplan, Ressourcenplan und Kostenplan sowie beim Projekt-Controlling. Gut berücksichtigt wird in der praxisorientierten Darstellung das Erstellen von Budgets, das Einplanen von Notgroschen und das Addieren von Zeitpuffern. Das Buch stützt sich auf viele instruktive Bildschirmabbildungen (Windows 7) und viele Schritt-für-Schritt-Anweisungen. Großes Fallbeispiel (betrifft EDV-Schulung). (2) (Klaus Barckow)
