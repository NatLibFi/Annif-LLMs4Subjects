Windows Server 2019 Automatisierung mit PowerShell – das Kochbuch : praxisorientierte Rezepte zur Automatisierung und Verwaltung von Administrationsaufgaben
PowerShell ist ein wichtiges Werkzeug zur Automatisierung, Konfiguration und Verwaltung von Arbeitsabläufen unter Windows Server. Das Buch hier ist somit eine gute Ergänzung zu Veröffentlichungen über Windows Server 2019 wie Th. Joos oder P. Kloep und ein guter Ersatz für das schon etwas ältere Buch von P. Monadjemi über PowerShell. Das Buch wendet sich an IT-Experten mit erheblichen Vorkenntnissen und setzt Zugriffsmöglichkeiten auf einen Windows 10- oder Windows Server 2019-Host mit aktivierter Hardware-Virtualisierung und Zugriff auf eine Cloud-Computing-Plattform voraus. Das Buch versteht sich als "Kochbuch" und stützt sich auf über 90 praxisnahe Beispiele. Didaktisch gut aufbereitet. Hilfreich ist in der Einleitung der ausführliche Absatz "Das Buch optimal nutzen". Viele Abbildungen. Bei jedem einzelnen Beispiel wird gezeigt, wie ein bestimmtes Feature von PowerShell funktioniert. Ergänzende Hinweise werden grafisch hervorgehoben. Auch auf Fehlerbehandlung wird gut eingegangen. (3)
