Microsoft Windows Server 2008 : Taschenratgeber für Administratoren ; [der praktische Begleiter für die tägliche Arbeit]
Zum neuen Betriebssystem Windows Server 2008 liegt bislang nur der Titel von Thomas Joos (ID 41/07) vor. Seine Beschreibung basiert noch auf der Beta-Version. Die vorliegende Ausgabe des durch zahlreiche Titel bekannten Autors (zuletzt zum Exchange Server 2007, ID 33/07) bietet in kompakter Form administrativ tätigen Personen alltagstaugliche Informationen zum deutlich veränderten System. Die Taschenratgeber sind nicht für das linerare Durcharbeiten gedacht, sondern weisen Referenzcharakter auf und dienen dem punktuellen Einstieg in bestimmte Fragestellungen. Ein differenziertes Inhalts- und ein umfangreiches Stichwortverzeichnis unterstützen diesen Ansatz. Schwerpunkte liegen bei der Konfiguration, den Gruppenrichtlinien, Fragen der Sicherheit, der Überwachung, der Systemwiederherstellung, dem Arbeiten mit dem Active Directory, der Verwaltung von Daten sowie der Netzwerkadministration. Hinzu kommen Tipps, Verfahrensempfehlungen und Beispiele. Zum Konzept gehören auch Schritt-für-Schritt-Anleitungen. Grundverständnis für Netzwerke und Windows Server-Betriebssysteme wird bei den Lesern vorausgesetzt.(3)
