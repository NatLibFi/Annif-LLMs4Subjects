Java 7 : das umfassende Training ; in 18 Stunden zum Java-Entwickler - inkl. Datenbanken, Streams, Threads, GUI-Programmierung, Servlets u.v.m. ; Video-Training
Java 7 ist die aktuelle Version einer oft eingesetzten objektorientierten Programmiersprache. Eine umfassende Darstellung zu Java 7 ist D. Louis (ID-B 4/12); Leser mit Vorkenntnissen seien auch auf B. Steppan (ID-A 50/11) hingewiesen. Das vorliegende 18-Stunden-Videotraining ist eine interessante Non-Print-Alternative zu den 2 Büchern. Der sog. Miniplayer ermöglicht es, die Workshops des Videotrainings parallel am eigenen Rechner nachzuvollziehen. Der Trainer beginnt mit der Installation und Konfiguration von Java und der Entwicklungsumgebung Eclipse.Es folgen grundlegende Programmierstrukturen (Variable, Fallunterscheidungen, Schleifen) und die Grundlagen der objektorientierten Programmierung (Objekte, Vererbung, Konstruktoren, Kapselung). Auch die Klassen der Standardbibliothek fehlen nicht. Gut berücksichtigt werden ebenfalls Werkzeuge und Hilfsmittel, z.B. das neue Filesystem NIO.2.API und die Frameworks Hibernate und Spring. So werden generische Programmierung, Datenbankzugriffe, parallele Programmierung und die Entwicklung grafischer Benutzeroberflächen für den Leser gut nachvollziehbar. (2)
