Java ist eine Sprache : Java lesen, schreiben und ausführen — Eine präzise und verständliche Einführung
Profitieren Sie von diesem einfachen Ansatz, Java zu erlernen, damit das Lesen, Schreiben und Ausführen von Programmen gelingt! Eine präzise und verständliche Einführung in die Programmierung, mit sauberen Begriffen, klarer und nachvollziehbarer Darstellung und korrekter Methodik. Berücksichtigt wird J2SE in der Version 5, so dass auch Generische Einheiten, Aufzählungstypen und viele andere wichtige Spracheigenschaften zum Zuge kommen. Unterstützende Daten stehen dem Leser kostenlos im Web zur Verfügung
