jQuery : das umfassende Handbuch ; [Grundlagen, Einsatz, Praxisbeispiele ; Responsive Webdesign, Plugins nutzen, jQuery UI ; Navigationen, Tabellen, Bildergalerien, Formulare ; mobile Anwendungen mit jQuery Mobile]
Die umfassende Einführung in das mächtige Framework jQuery vermittelt notwendige Grundlagen und bietet zahlreiche Beispiele und Lösungen. (LK/HB: Isigkeit)
Beim Framework jQuery handelt es sich um eine Erweiterung der Skriptsprache JavaScript. Es unterstützt die Arbeit der Web-Anwendungsentwickler und wird auf vielen Webseiten eingesetzt. Die beiden Autoren haben eine sehr umfangreiche und praxisorientierte Einführung erarbeitet, die jetzt in einer neuen, aktualisierten Auflage vorliegt (zuletzt 2. Auflage: ID-B 43/11). Den behutsamen Veränderungen in den 4 Anfangskapiteln folgen ein neues Unterkapitel zur Skriptorganisation und völlig neu eingerichtete Kapitel zum jQuery User Interface (jQuery UI), zur Plug-in-Entwicklung und zum Responsive Webdesign. Letzteres ergänzen eine Überarbeitung des Kapitels zu jQuery mobile und ein Zusatzkapitel für Fortgeschrittene zu jQuery-Build. In die Überarbeitung einbezogen wurde auch das Abschlusskapitel zum Testen von Anwendungen. Obsolete Inhalte wurden entfernt. Alle Kapitel enden mit Zusammenfassungen. Eine sinnvoll gefüllte DVD in der Anlage. In dieser Form ist die Bezeichnung Standardwerk zu jQuery gerechtfertigt. Vorausgabe ersetzen. Vor R. Steyer (ID-B 15/14). (2) (LK/HB: Isigkeit)
