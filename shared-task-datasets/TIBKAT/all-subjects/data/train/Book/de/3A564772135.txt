Microsoft Windows Server 2008 : die Neuerungen im Praxiseinsatz
Windows Server 2008 ist die aktuelle Version des Serverbetriebssystems von Microsoft. Die Unterschiede zur vorigen Version 2003 sind beträchtlich. Der Autor hat die Neuerungen von Windows Server 2008 bereits an Hand einer Beta-Version beschrieben (BA 11/07). Das vorliegende Buch ist eine überarbeitete Auflage, die sich aber auf die finale Version von Windows Server 2008 stützt. Die Gliederung des Buches folgt im Wesentlichen der Beschreibung für die Beta-Version. Sehr häufig sind Textübereinstimmungen festzustellen. Neu hinzugekommen oder ausführlicher gehalten sind Ausführungen über Server Core, Windows-Bereitstellungsdienste, die Windows Power Shell sowie Datensicherheit und Datenwiederherstellung. Der Praxisbezug wurde verstärkt. Neu sind auch zusätzliche Workshops und die Zusammenfassungen zu den einzelnen Kapiteln. Schon die Beschreibung der Beta-Version gibt einen guten Überblick. So muss offen bleiben, ob bei dem Wunsch nach zusätzlicher Literatur nicht auf die ausführliche Darstellung von W. Stanke (ID 21/08) verwiesen werden sollte. (3)
