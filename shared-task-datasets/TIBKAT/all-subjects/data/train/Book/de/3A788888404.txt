JavaScript Programmierung von Kopf bis Fuß : [ein Lernbuch zur JavaScript-Programmierung]
Unterstützt durch ein abwechslungsreiches Layout werden Anfänger mit Vorkenntnissen in HTML und CSS in die Programmierung mit JavaScript eingeführt. (LK/LEV: Junker)
Neben HTML und CSS gehören Kenntnisse in JavaScript zu den wichtigsten Grundlagen für die Programmierung von Webseiten. Seit der vorigen Ausgabe (ID-G 34/08 unter M. Morrison) ist JavaScript bei Version 1.8.5 angelangt, kompatibel zum aktuellen Standard ECMAScript 5, der auf allen modernen Browsern umgesetzt wird. Das neue Autorenduo hat den Inhalt stark umgekrempelt, um knapp 50 Seiten und 2 Kapitel erweitert und unter anderem auf die gestiegene Bedeutung der App-Programmierung angepasst. Mithilfe des auf kognitionswissenschaftlichen Erkenntnissen beruhenden sehr sprunghaften Layouts, vielen Beispielen und Übungen lernen die Anwender die wichtigsten Grundlagen der Programmiersprache und die Entwicklung von browserübergreifenden Skripten. Auch durch JavaScript entstehende Sicherheitslücken werden behandelt und Tipps zu deren Vermeidung gegeben. Zur Erschließung dienen ein ausführliches Inhaltsverzeichnis und ein Stichwortindex. Alternativ zu konventionellen Titeln wie C. Wenz (ID-B 22/14) oder R. Steyer (ID-B 24/14), die stärker auf HTML5 eingehen. Die alte Ausgabe verbrauchen. (2) (LK/LEV: Junker)
