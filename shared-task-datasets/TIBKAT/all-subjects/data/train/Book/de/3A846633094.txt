Kompaktkurs C# 6.0
C [sharp] ist eine von Microsoft entwickelte Programmiersprache, die eine wichtige Rolle im Rahmen der .NET-Strategie spielt. Der Autor des vorliegenden Buchs ist bekannt durch Kompaktkurse zu Vorgängerversionen der aktuellen Version C [sharp] 6.0. Eine gute Kurzeinführung zur Version 5.0 ist auch J. Albahari. Das Buch hier legt den Schwerpunkt auf die Aktualisierungen der aktuellen Version (Zugriff über Referenzen, Ausnahmefilter, ...). Bücher über die Vorgängerversion können noch im Bestand verbleiben, sollten aber bei Ersatzbeschaffungen ersetzt werden. Angesprochen werden Leser mit Vorkenntnissen, etwa in Java oder C++. Am Ende eines jeden Kapitels finden sich Übungsaufgaben (mit Musterlösungen im Internet). Der Autor beginnt mit einer kurzen Einführung in das .NET-Framework und die wesentlichen Unterschiede zwischen C [sharp] und Java und kommt dann zu den Sprachelementen von C [sharp]. Sehr hilfreich sind auch 3 Fallstudien (grafische Benutzeroberfläche, Webservices, dynamische Webseite). Auch zum Selbststudium geeignet
