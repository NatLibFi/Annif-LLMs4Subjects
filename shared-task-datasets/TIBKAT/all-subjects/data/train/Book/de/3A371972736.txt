Professionelle Softwareentwicklung mit PHP 5 : Objektorientierung - Entwurfsmuster - Modellierung - fortgeschrittene Datenbankprogrammierung
In der für das dritte Quartal 2003 angekündigten Version 5 überwindet PHP die technischen Beschränkungen des Objektmodells, zu denen u.a. das Fehlen von privaten Attributen und Methoden sowie Probleme beim Umgang mit Objektreferenzen zählten. Mit den neuen Sprachmerkmalen und der Script Running Machine SRM (dem ersten Application Server für PHP) schließt die PHP-Plattform zu anderen Technologien auf. Das Buch beschreibt erfahrenen PHP-Entwicklern den Weg in die neue, objektorientierte PHP-5-Welt. Programmierer, die bislang mit objektorientierten Programmiersprachen wie C++ oder Java gearbeitet haben, lernen hier, wie sie die ihnen vertrauten Konzepte und Entwurfsmuster mit PHP umsetzen können
