Programmierung in Google Go
Go ist eine 2009 von Mitarbeitern von Google entwickelte neue Programmiersprache, mit Möglichkeiten einer hardware-nahen Softwareentwicklung, die sich an C/C++ und Perl orientiert. Go ist für Linux und Mac OS X (ab Version 10.5) verfügbar; neuerdings steht es auch in einer experimentellen Version für Windows (ab Windows XP) bereit, bei der allerdings einige wichtige Tools fehlen. - Die Autoren sprechen Leser mit Vorkenntnissen in einer objektorientierten Sprache an, die bei Programmiersprachen "auf der Höhe der Zeit bleiben wollen". Zusammengefasst werden zunächst Gründe für die Entwicklung von Go (leicht zu lernen, Beschränkung auf das absolut Notwendige, gute Berücksichtigung parallelisierter Hardware, zunehmende Bedeutung des Server-Computing, ...), erläutert die Installation (unter allen Betriebssystemen). Es folgt gestützt auf viele lauffähige Beispiele eine Einführung in das Programmieren mit Go. Gut berücksichtigt wird die Nutzung von C-Bibliotheken unter Go. - Auf Begleit-CD eine bootbare Linux-Distribution mit einer aktuellen Go-Installation. Beispiele können so direkt nachvollzogen werden. (2) (Klaus Barckow)
