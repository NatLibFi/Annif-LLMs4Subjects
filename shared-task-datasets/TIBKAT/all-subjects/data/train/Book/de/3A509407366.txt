Objektorientierte Programmierung mit Visual Basic 2005 : [eine moderne Einführung ; Klassen, Objekte, Vererbung und Polymorphie praktisch angewendet ; Übungen mit Musterlösungen zu jedem Kapitel ; auf CD: Visual Basic 2005 Express Edition]
Eine weitere Publikation zur neuesten Version der Programmiersprache, die in dieser Fassung passend zur neuen Generation des .NET Frameworks 2.0 eingesetzt werden kann. Das Buch ist sehr systematisch aufgebaut. Eingangs wird das VB-Konzept und die Oberfläche erläutert. Anschließend folgen einige Kapitel zur prozeduralen Programmierung. Sie bilden die Grundlage für den Neueinsteiger. Alsdann folgen die Konzepte der Objektorientierung, die Schritt für Schritt an einem durchgängigen Beispiel entwickelt werden. Jedes Kapitel endet mit einer Zusammenfassung sowie mit Fragen und Übungsbeispielen. Die Lösungen befinden sich im Anhang. Das Stichwortverzeichnis ist recht knapp gehalten. Eine didaktisch geschickt aufgebaute und gut verständliche Einführung. Sie richtet sich an Programmieranfänger oder Umsteiger von älteren Versionen. Auf CD-ROM VB 2005 Express Edition, die ein sofortiges Nachvollziehen der Beispiele ermöglicht. Gut einsetzbar neben Michael Halvorson (BA 5/06) und dem weiterführenden Titel von Peter Bloch (BA 6/06). (2)
