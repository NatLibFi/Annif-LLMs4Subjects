Einstieg in Java 6
Neuversionen der plattformunabhängigen Programmiersprache Java erscheinen laut Hersteller Sun im Rhythmus von 18 Monaten, seit Dezember 2006 heißt die aktuelle Version Java 6 bzw. genau genommen Java 2 Standard Edition 6.0 und ist Nachfolger von Java 5, auf das seit September 2004 der Buchmarkt mit zahlreichen Neuerscheinungen reagierte. Notwendigerweise hat Autor Steppan nun auch seine grundlegende Einführung in die Java-Programmierung von 2005 (BA 3/05) überarbeitet und aktualisiert ohne allerdings vom bewährten didaktischen Konzept abzuweichen. Weiterhin werden Programmiertheorie, Grundlagen der Objektorientierung, Sprachelemente und -konzepte gleichwertig mit praktischen Programmierbeispielen und Praxisinformationen zu den zahlreichen Java-Tools vermittelt. Der gut strukturierte, bestens zum Selbststudium geeignete Band kann Programmiereinsteigern weiterhin vorrangig empfohlen werden. Der Vorgänger ist zusammen mit der übrigen Java-5-Literatur weiterhin bestandsrelevant, Java 6 sollte aber in allen EDV-Beständen mit mehreren Titeln abgedeckt sein - bis wohl Ende 2008 Java 7 ins Haus steht!
