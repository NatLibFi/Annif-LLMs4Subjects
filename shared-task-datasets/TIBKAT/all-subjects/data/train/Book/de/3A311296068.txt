CORBA für Dummies : gegen den täglichen Frust mit CORBA
CORBA (Common Object Request Broker Architecture) bezeichnet die Spezifikation eines Standards mit dem Ziel, Objekte, die auf verschiedenen Plattformen laufen und die in verschiedenen Sprachen geschrieben sind, zusammenzubinden, um eine neue Applikation zu entwickeln. Die Autoren wenden sich an Einsteiger, die allerdings über Vorkenntnisse in der objektorientierten Programmierung und in C++ oder Java (als Zielsprachen von CORBA) verfügen sollten. Sie beschreiben in lockerer, aber doch exakter Darstellung Grundkonzepte und Arbeitstechniken für die aktuelle Version CORBA 2.2. Sie gehen ausführlich auf zwei für CORBA wichtige Thematiken ein - ORBs (Object Request Brokers), die den Kern von CORBA bilden und dafür sorgen, dass Objekte (Anwendungen) miteinander kommunizieren können, egal wo sie sich befinden und auf welchem System sie laufen, und auf IDL (Interface Definition Language), eine Sprache, die Schnittstellen zwischen verteilten Objekten beschreibt. Auf CD-ROM Muster-ORBs, die 60 Tage kostenfrei benutzt werden können. Aufzeigen von Entwicklungstendenzen. Aktueller als J. Rosenberger (ID 40/98). (3) (Klaus Barckow)
