SQL Server 2008 R2 - der schnelle Einstieg : Abfragen, Transact-SQL, Entwicklung und Verwaltung ; [für SQL Server 2008 und 2008 R2]
Rezension: Mit der Bezeichnung R2 pflegt Microsoft Produkte zu bezeichnen, die zwar keine echte Neuversion eines Programmes, aber eine wesentlich verbesserte Ausgabe, ein Zwischenrelease darstellen. Das vor allem im kommerziellen Bereich eingesetzte relationale Datenbanksystem SQL Server stellte der Autor zuletzt anlässlich des Umstiegs auf Version 2008 vor (BA 8/09; noch gut brauchbar). Gliederung und Darstellungsweise hat Konopasek vom Vorgänger übernommen, aber schon der Mehrumfang von 60 Seiten zeigt, dass einiges an Neuerungen zu berücksichtigen war. Die bestehen vor allem aus erweiterten Business-Intelligence-Lösungen und einem neuen Tool zur Administration von Datenebenen. Alle Neuerungen sind in einem eigenen Kapitel zusammengefasst und bei der Behandlung im systematischen Hauptteil durch einen entsprechenden Icon kenntlich gemacht, was das Buch für Umsteiger von früheren Ausgaben besonders interessant macht. Die Neuausgabe kann sinnvoll auch neben dem Vorgängerband angeboten werden, Paralleltitel anderer Verlage (z.B. Galileo Press) sind aber angekündigt und können eventuell abgewartet werden. (2)
