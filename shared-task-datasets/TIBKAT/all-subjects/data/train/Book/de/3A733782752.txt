Grundkurs gutes Webdesign : alles, was Sie über Gestaltung im Web wissen sollten ; [mit Schriften, Farben und Grafiken gestalten ; CSS-Layouts, Navigationen, Icons, Buttons ; Usability, Webstandards und Responsive Webdesign]
Dank Vierfarbdruck und gutem Layout macht der Titel für den Preis einen sehr hochwertigen Eindruck. Und erfreulicherweise setzt sich die äussere Qualität auch im Inneren fort. Man darf den Titel wörtlich nehmen. Es geht um das Design im Sinne von Gestaltung. Das unterscheidet das Buch von ähnlich lautenden Titeln, die oft aber Einführungen in HTML und CSS mit Randbemerkungen zur Gestaltung darstellen. Hier werden HTML- und CSS-Kenntnisseund daneben auch Photoshop-Wissen explizit vorausgesetzt. Das Thema sind die Regeln (u.a. aus der Gestaltpsychologie), die bei der Gestaltung von Websites befolgt werden müssen. Behandelt werden: Website-Konzeption, Usability, Layout und Komposition, Typografie, Farbe, Grafiken und Bilder, Test und Optimierung. fragen des responsive Webdesigns werden angemessen berücksichtigt. Das Buch führt niedrigschwellig, aber fachlich auf höherem Niveau ins Thema ein. Es kann damit hervorragend neben den bekannten Titeln zur Einführung in HTML und CSS verwendet werden (1)
