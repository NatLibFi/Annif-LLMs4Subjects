MySQL : das offizielle Handbuch ; [das Insider-Wissen der MySQL-Entwickler ; wie Sie MySQL optimieren, debuggen und erweitern ; mit umfassender MySQL-Sprachreferenz + Tutorial]
MySQL ist ein relationales Datenbankprogramm, das die wichtigeren Befehle der standardisierten Datenbankabfragesprache SQL (Structured Query Language) unterstützt. MySQL ist für Privatanwender kostenfrei erhältlich; es benötigt längst nicht so viele Ressourcen wie kommerzielle Datenbanksysteme. MySQL ist als schnelles System für Applikationen mit hoher Auslastung besonders geeignet. Das vorliegende Buch basiert auf dem offiziellen Online-Manual von MySQL; zusätzliche Rückmeldungen von Anwendern wurden berücksichtigt. Behandelt wird das stabile Release 3.23.x; Änderungen des zukünftigen Release 4.0.x werden erwähnt. Das Buch ist somit aktueller als die umfassende Darstellung von R. Yarger (ID 38/00 - kann noch im Bestand verbleiben). Einen ersten Einstieg gibt R. Däßler (BA 10/01). Schwerpunkte liegen bei Sicherungsmechanismen und bei Optimierungsmöglichkeiten. Gut berücksichtigt werden auch Stärken und Schwächen der einzelnen Tabellentypen, die Fehlerbehandlung, Erweiterungsmöglichkeiten für MyS und die Unterschiede zwischen MySQL und dem SQL-Standard. Umfassender Referenzteil mit vielen Beispielen. (2) (Klaus Barckow)
