Hitchhiker's Guide für Python : best Practices für Programmierer
Das vorliegende Buch ist aus einem Gemeinschaftsprojekt der Python-Community entstanden und soll das Interesse für die Programmiersprache Python wecken. Es geht weit über Einführungen in Python für Anfänger (in Python 3) wie z.B. M. Weigend oder B. Klein hinaus und behält auch Python 2 im Blickpunkt. Gleich zu Beginn wird festgestellt, dass die Abkehr vom alten Standard Python 2.7 bei Weitem nicht von allen Entwicklern mitgetragen wird und dass es Sicherheits-Updates für Python 2.7 bis 2020 geben wird. Ziel des Buchs ist, das geballte Insiderwissen der Python-Community und ihre bevorzugten Werkzeuge (insbesondere Python-Bibliotheken) vorzustellen. Die Autoren beginnen mit der Installation von Python unter verschiedenen Betriebssystemen, mit Interpretern und Entwicklungsumgebungen. Es folgen Bibliotheken (auch Netzwerkbibliotheken und Bibliotheken zur Datenpersistenz und Datenmanipulation). Gut berücksichtigt werden auch Werkzeuge zur Bild- und Audioverarbeitung). Viele Abbildungen, oft (kommentierte) Codeabschnitte. Ebenso Hinweise auf typische Fallstricke
