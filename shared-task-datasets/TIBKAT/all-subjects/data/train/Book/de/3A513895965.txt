Java 6 : Praxis der objektorientierten Programmierung ; [auf Vista getestet]
Java 6 ist die aktuelle Version einer weitverbreiteten Programmiersprache, die insbesondere für das Internet von großer Bedeutung ist. Die aktuelle Version unterscheidet sich von der Vorgängerversion beträchtlich. Einführungen für Einsteiger sind D. Louis (BA 5/07) oder B. Steppan (BA 4/07). Eine umfassende Darstellung für Fortgeschrittene ist D. Frischalowski (BA 5/07). Das vorliegende Buch ist eine interessante Alternative zu dem letztgenannten Werk. Es stellt nicht ganz so hohe Ansprüche an den Leser; auch bringt es eine größere Anzahl von Codebeispielen, die direkt als Bausteine für eigene Entwicklungen eingesetzt werden können. Die wichtigsten Neuerungen von Java 6 finden sich in einer Übersicht zu Anfang des Buches. Ein Schwerpunkt liegt bei der Klassenbibliothek von Java 6; alle Klassen mit Bedeutung für den Alltagsbetrieb eines Programmierers - etwa 90 % - werden ausführlich erläutert. Gut berücksichtigt werden auch grafische Benutzeroberflächen und Datenbankzugriffe. Aufgaben fehlen leider. Die Autoren sind bekannt durch Bücher zu älteren Versionen von Java (zuletzt BA 3/05 - Java 5). (2)
