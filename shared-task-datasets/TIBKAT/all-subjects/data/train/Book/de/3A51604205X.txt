Oracle Database 10g Express Edition : der schnelle Einstieg
Heitsiek (zuletzt "Das Einsteigerseminar C++", BA 06/05) bietet eine gelungene Einführung für Oracle-Anfänger. Mit der Expresss Edition bietet Oracle eine eingeschränkte Version seines Datenbankmanagementsystems an, analog dem SQL Server 2005 Express des Konkurrenten Microsoft. Die Beschränkung liegt lediglich bei einigen Hardwaremerkmalen, bietet ansonsten aber volle Funktionalität. Beginnend mit einer allgemeinen Einführung über Datenbanken, werden in den folgenden Kapiteln Informationen zur Installation und Administration angeboten. Ausführliche Kapitel über SQL und den Datenbankzugriff über Java erläutern die tägliche Arbeit mit der Datenbank. Die beiliegende CD-ROM enthält die Oracle 10g Express Edition, alle SQL Skripte sowie Java-Beispielprogramme. Für Schüler, Studenten oder Entwickler, die mit Oracle beginnen möchten, bietet dieses Buch eine gelungende Einführung. Für erfahrende Benutzer bzw. für tiefergehende Informationen dient u.a. "Oracle 10g" (ID 6/05). (2 S)
