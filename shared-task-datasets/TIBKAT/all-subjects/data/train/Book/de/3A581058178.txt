Effizienter Programmieren mit Visual Studio : [250 Tipps, um Ihre Produktivität zu verbessern]
Visual Studio ist eine integrierte Entwicklungsumgebung von Microsoft für Visual Basic, C++/CLI und C[sharp], also für .NET-Sprachen. Über Möglichkeiten einer Individualisierung von Visual Studio (z.B. mithilfe von Add-Ins) und starke Vereinfachungen der objektorientierten Programmierung mit Visual Studio berichtet C. Skibo (ID 11/06). Behandelt wird dort die Vorgängerversion Visual Studio 2005, sodass die neuen interessanten Möglichkeiten der aktuellen Versionen 3.0 und 3.5 des .NET-Frameworks noch nicht berücksichtigt sind. Die Autorin des vorliegenden Buches ist Mitglied des Visual Studio Produkt Teams und musste alle Features und Möglichkeiten von Visual Studio, auch in der aktuellen Version 2008, ausprobieren. Ihre Erfahrungen fast sie hier in 250 Tipps zusammen. Schwerpunkte liegen beim Editor, beim Suchen und beim Layout. Aber auch das Debugging wird gut berücksichtigt. Großer Wert wird darauf gelegt, dass die Vorschläge einfach und schnell umgesetzt werden können; das Programmieren soll ja effizienter werden. - Besondere Vorkenntnisse sind beim Leser nicht erforderlich.
