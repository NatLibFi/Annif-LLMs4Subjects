Das Einsteigerseminar Java mit Eclipse
Der einschlägig bekannte Autor hat seine seit 1996 innerhalb der Reihe stets neuen Versionen angepasste Einführung in die Javaprogrammierung anlässlich der Einführung von Java 6 (vgl. hierzu auch die kürzlich besprochenen Titel von C. Ullenboom: ID 4/07; B. Steppan: BA 3/07) erneut überarbeitet. Wesentliche Änderung gegenüber dem hier zuletzt besprochenen Vorgängertitel zu Java 2 (BA 8/02; kann verbraucht werden) ist die wesentlich kompaktere Einführung in die Java-Programmierung über die Kommandozeile zugunsten der nun breiten Raum einnehmenden Programmierung unter der Oberfläche Eclipse. Letztere ist als frei erhältliches Tool, das inzwischen aktuell als Version 3.2 vorliegt, nicht mehr aus der komfortablen Java-Programmierung wegzudenken. Wie von der Reihe gewohnt werden dem vorkenntnisfreien User in einem Lern-, Übungs- und Anwendungsteil die wichtigsten Informationen für den Ersteinstieg sehr einprägsam vermittelt. Zur Vertiefung allerdings sind umfassendere Einzeltitel zu Java und/oder Eclipse unabdingbar. (1)
