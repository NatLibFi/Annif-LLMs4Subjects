Grundkurs MySQL und PHP : so entwickeln Sie Datenbanken mit Open-Source-Software
Dynamische Webseiten, wie z.B. Bestellformulare oder Diskussionsforen, werden heute meist mit Datenbanken realisiert. Hilfsmittel hierfür ist u.a. die weitgehend kostenfrei erhältliche Open-Source-Software MySQL und PHP. Das Thema ist bereits von S. Letzel (BA 11/02 - kann im Bestand verbleiben) abgehandelt worden. Das vorliegende Buch gibt den aktuellen Stand der Technik wieder: Hauptziel ist eine ausführliche praxisnahe Einführung in die sog. LAMP-Konfiguration (Linux - als Betriebssystem, Apache - als Serversoftware; MySQL - als Datenbankmanagementsystem; PHP - als moderne Skriptsprache). Aber auch auf eine Implementierung unter Windows wird eingegangen. Zur Verdeutlichung wird in Übungsaufgaben schrittweise eine Datenbank aufgebaut (Datenbank für Literatur zu Datenbanken) und mit einer Web-Schnittstelle ausgestattet. Diese Datenbank wird abschließend auch noch mit einem Recherche-Interface versehen und ermöglicht so Literaturrecherchen. Auf Sicherheitsaspekte (Vergabe von Rechten, interne Verwaltung) wird im notwendigen Umfange eingegangen. Grundkenntnisse in C sind hilfreich. (3) (Klaus Barckow)
