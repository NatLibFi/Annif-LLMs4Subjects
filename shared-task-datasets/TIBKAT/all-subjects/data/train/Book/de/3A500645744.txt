HTML Handbuch
Standardwerk zur Auszeichnungssprache HTML. Behandelt alle wichtigen Aspekte einschließlich CSS, JavaScript und DOM. (LK/HB: Isigkeit)
Bei der Auszeichnungssprache HTML handelt es sich nach wie vor um eine Basistechnologie für die Webseitenerstellung. Basierend auf der Internetdokumentation SelfHTML erschienen in der Vergangenheit gedruckte Ausgaben des vorliegenden Handbuchs ("HTML 4.0 Handbuch", BA 4/99, zuletzt: Wolfgang Nefzger: "HTML 4, XHTML", BA 12/02). Das bewährte Autorenduo bietet mit dieser aktualisierten Ausgabe die meistgelesenen Kapitel der ständig verbesserten SelfHTML-Site. Das Standardwerk dokumentiert neben HTML auch CSS, JavaScript und Document Object Model (DOM). Eine randvolle DVD mit vorzüglicher Software begleitet das Buch. Diese im Preis attraktive Studienausgabe, die auf eine feste Bindung verzichtet, sollte die 1999er-Ausgabe ersetzen. (1)
