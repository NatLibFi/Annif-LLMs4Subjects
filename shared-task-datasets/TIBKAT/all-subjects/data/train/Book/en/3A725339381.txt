Decompiling Android : [discover how Android apps are compiled and built to secure your data and build better apps]
Protect your code by examining the reasons why Android apps can be decompiled to recover their source code. This book shows first hand how good and bad Android apps are constructed and how to build successful apps. An in depth examination of the Java and Android class file structures is provided along with how easy it could be to download an Android app and decompile it into its original Java source, HTML5 and CSS code. This book shows how to stop this from occurring with apps. Learn to protect code and eliminate back doors that can allow people to potentially obtain sensitive information from apps
