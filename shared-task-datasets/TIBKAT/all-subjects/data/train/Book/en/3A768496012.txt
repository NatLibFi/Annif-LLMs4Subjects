PhoneGap build : developing cross platform mobile applications in the cloud
"This book begins by explaining the differences between existing mobile platforms, the different types of browsers they support, programming languages, and IDE required to develop apps for each of them. Next, it explains how PhoneGap makes the task of developing cross platform mobile apps easier. Starting with small applications the book builds towards explaining advanced concepts, exploiting different APIs, and the different methods for real-world applications. The book provides readers with the understanding to develop feature-rich mobile applications that can run on the cloud and support different platforms"--
