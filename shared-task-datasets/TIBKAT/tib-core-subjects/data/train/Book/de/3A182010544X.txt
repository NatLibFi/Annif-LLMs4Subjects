Java für Dummies
Sie wollen in Java einsteigen oder Ihre Kenntnisse erweitern? "Java für Dummies" ist gleichzeitig Lehrbuch und unverzichtbares Nachschlagewerk für alle Java-Programmierer. Basiswissen zur objektorientierten Programmierung wird genauso vermittelt wie das Prinzip der Wiederverwendbarkeit von Programmbausteinen. Ausserdem lernen Sie, wann Variablen innerhalb oder ausserhalb von Methoden deklariert werden sollten, wo die Grenzen von Arrays liegen und wie Code mit Exceptions absturzsicher gemacht wird. Diese Auflage von "Java für Dummies" berücksichtigt die Neuerungen der Version Java 17
