Windows XP dirty Tricks
J. van Heyl ist durch ein Buch bekannt, das Leser auf Tuningmöglichkeiten für Windows durch Eingriffe in die Registry hinweist (BA 12/02 - noch aktuell). Im vorliegenden Buch behandelt er Windows XP allgemein und gibt, sachlich gegliedert, viele Tipps für einen schnelleren und sichereren Betrieb. Auch auf das neue Service Pack 2 wird eingegangen; Vorteile und Schwächen werden herausgearbeitet. Beim Leser werden nur normale Kenntnisse im Umgang mit Windows erwartet. Schritt-für-Schritt-Anleitungen machen die Umsetzung eines jeden Tipps leicht. Auf eventuelle Gefahrenpunkte wird jeweils hingewiesen; Rettungsmaßnahmen für den Notfall werden gleich mitgeliefert. Grenzen der Legalität werden gelegentlich berührt, aber auch hier entsprechende Hinweise. Hilfreich sind die Hinweise auf kostenfreie Tools (mit Downloadadressen), die oft leistungsfähiger sind als entsprechende Produkte von Microsoft. Aber auch auf die versteckten Tools im Betriebssystem, die nur schwer zu finden sind, das Arbeiten aber wesentlich erleichtern, wird hingewiesen. Spezielle Probleme für PC-Spiele werden ausführlich diskutiert. (1)
