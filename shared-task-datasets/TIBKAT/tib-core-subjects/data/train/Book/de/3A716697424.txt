Microsoft Windows Server 2012 - das Handbuch
Thomas Joos, ausgewiesener IT-Spezialist, veröffentlicht immer wieder Bücher zur Servertechnologie (zuletzt "Exchange Server 2010"; ID-G 34/12). Jetzt hat er sich der aktuellen Version des Server-Betriebssystems Windows Server 2012 angenommen. Der Inhalt baut auf der Vorgängerausgabe zu Windows Server 2008 R2 auf (ID-G 34/11), entstanden ist jedoch ein neu strukturiertes und mit veränderten Inhalten versehenes Werk. Die umfangreichen Verbesserungen und Neuerungen, u.a. im Bereich Virtualisierung und Zusammenarbeit von Servern im Netzwerk, werden im 1. Kapitel vorgestellt und erfahren in den nachfolgenden Kapiteln eine detaillierte Beschreibung. Auch die schmalere Version Windows Server 2012 Essentials, für kleinere Unternehmen, wird berücksichtigt. Mit fast 1.300 Seiten fällt der Umfang etwas bescheidener aus, dennoch werden alle Funktionalitäten ausführlich und verständlich beschrieben. Angesprochen werden IT-Profis, die sich mit dem Betriebssystem vertraut machen wollen bzw. Umsteiger von älteren Systemen. Die beigefügte CD-ROM enthält eine E-Book-Fassung des Titels. Neben W.R. Stanek (ID-G 8/13). (3)
