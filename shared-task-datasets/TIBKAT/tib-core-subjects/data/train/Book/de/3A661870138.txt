Einstieg in Eclipse 3.7 : aktuell zu Indigo [und Java 7 ; effiziente Java-Entwicklung mit Eclipse ; Plug-ins, Web- und RCP-Anwendungen erstellen ; inkl. Refactoring, Debugging, Subversion, CVS u.v.m.]
Eclipse ist eine Open-Source-Entwicklungsumgebung für verschiedene Programmiersprachen, vor allem aber für die Java-Technologie. Der Autor hat seine den Ansprüchen von Einsteigern wie Fortgeschrittenen gerecht werdende Einführung in Version 3.5 (BA 2/10) auf den Stand der Version 3.7 hin aktualisiert. Wichtigste Neuerung ist die Unterstützung von Java 7. Die derzeit einzige aktuelle Einführung in Eclipse konzentriert sich weiterhin allein auf Java als grundlegende Programmiersprache und beschreibt wie bereits in den letzten beiden Vorauflagen (können verbraucht werden) sehr ausführlich die Entwicklung von und die Arbeit mit Plug-ins. Solide Grundkenntnisse in Java werden vorausgesetzt. Empfiehlt sich dringend zur Ergänzung des Vorgängers und zum Ersatz von Titeln zu älteren Eclipse-Versionen. In größeren Beständen sollte an ausreichende Staffelung gedacht werden, da im Gegensatz zu früheren Eclipse-Versionen nicht mit weiteren Vergleichstiteln zu rechnen ist. (2) (LK/M: Zimmermann)
Praxisinformation zur Arbeit mit der Entwicklungsumgebung Eclipse in Version 3.7 für Programmierer mit Java-Vorkenntnissen. (LK/M: Zimmermann)
