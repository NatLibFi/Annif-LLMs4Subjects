Java - exemplarisch : learning by doing
Main description: Dieses Lehrbuch soll vermitteln, wie Java als Erstsprache unterrichtet und gelernt werden kann. Dem Buch liegt ein didaktisches Konzept zu Grunde, das sich beim Selbststudium und in der Kurs- und Unterrichtspraxis über Jahrzehnte bewährt hat. Ausgehend von den sechs Thesen: 1.Unterrichte eine moderne, weitverbreitete Programmiersprache; 2. Teile den Unterricht in kleine, logisch aufeinanderfolgende Lernschritte ohne Vorwärtsbezüge ein; 3. Reduziere die Komplexität durch Auslassungssünden; 4. Führe jedes Konzept mittels lauffähiger Programme ein; 5. Verwende didaktisch konzipierte Klassenbibliotheken; 6. Vermittle die Denkweise der OOP von Anfang an; führt der Autor in die Sprache ein und ermöglicht es dem Studierenden, die Grundkenntnisse von Java zu erlernen und dadurch das Fundament für den tieferen Einstieg in Java oder jede andere Objektoriente Sprache zu legen.
