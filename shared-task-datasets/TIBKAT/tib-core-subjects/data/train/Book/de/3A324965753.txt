Java und XML
Java und XML sind wegen ihrer Plattformunabhängigkeit beide für sich genommen mächtige Technologien, zusammen aber bilden sie die ideale Lösung zur Entwicklung webbasierter Anwendungen. Mit Java kann man portablen Code entwickeln, ,it XML portable Daten generieren. Ihre gemeinsamen Eigenschaften wie Erweiterbarkeit, Wiederverwendbarkeit und globale Sprachunterstützung (Unicode) machen sie in ihrem Zusammenwirken so effektiv. Das Buch behandelt: a) die Grundlagen von XML b) die Benutzung von SAX-APIs und DOM-APIs zum Parsen von XML c) das Entwickeln neuer Dokumenttypen mit DTDs und XML Schema d) das Schreiben von Programmen, die XML- Daten generieren e) die Konvertierung von XML in verschiedene andere Formate mit XSL/T f) die Programmierung von Applikationen mit XML-RPC g) den Einsatz von Frameworks wie Apache Cocoon h) die Verwendung von XML als Konfigurationssprache i) die Programmierung von Business-to-Business-Applikationen mit XML j) das Entwickeln von Informations-Channels mit RSS und von dynamischem Inhalt mit XSP.
