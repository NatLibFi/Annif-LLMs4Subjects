Einfach Java : gleich richtig programmieren lernen
Der Autor führt die Leser*innen gut verständlich von den Grundlagen der Programmierung wie z.B. Datentypen und Variable über Grundlagen der Sprache Java und ihre theoretische Basis Objektorientierung bis hin zu komplexen Konstruktionen wie Containerklassen, die Objektreferenzen speichern und Lambdas als Behälter für Funktionscode. Es gibt bereits einige gut verständliche Einführungen in Java - z.B. K. Günster (2017) auf Basis Version 9. Das Besondere am vorliegenden Buch ist das erklärte Ziel des Autors, die Leser*innen anzuleiten von Beginn an übersichtlichen und gut strukturierten Code zu schreiben. Das erreicht er zum einen durch die Darstellung von fortgeschrittenen modernen Java-Konzepten wie komplexe Vererbungsmechanismen etc., die den Code vereinfachen und zum andern durch die Empfehlung eines guten Programmierstils, dem er ein Kapitel widmet. Das Buch basiert auf der Java-Version 11, der aktuellen Long-Term-Support-Version. Folgeversionen beinhalten einige hilfreiche Syntax- und Application-Interface-Erweiterungen, die der Autor teilweise in Beispielen nutzt (bis Version 17)
