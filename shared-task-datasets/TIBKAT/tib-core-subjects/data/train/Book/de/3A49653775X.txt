Microsoft ADO.NET 2.0 Schritt für Schritt
ADO.NET ist die Datenbankzugriffskomponente des .NET-Framework von Microsoft, die als Bestandteil der Entwicklungsumgebung Visual Studio 2005 jetzt in der Version 2.0 vorliegt. Die Unterschiede gegenüber der vorhergehenden Version 1.0 sind beträchtlich (neue Objekte, Methoden und Eigenschaften), sodass die Einführung der Autorin in die Version 1.0 (ID 34/03) überholt ist. Angesprochen werden Leser, bei denen Grundkenntnisse in .NET-Framework und in Visual Studio 2005 vorausgesetzt werden. Das Buch arbeitet in seinen Beispielen und Übungen wahlweise mit Visual Basic oder Visual C@So werden auch umfangreichere Kenntnisse in einer dieser Sprachen vorausgesetzt. Das didaktisch gut gestaltete Buch führt praxisnah Schritt für Schritt in die wesentlichen Arbeitstechniken von ADO.NET 2.0 ein. Viele Hinweise für Umsteiger von ADO.NET Version 1.0 und sogar von früheren Versionen von ADO. Auch das Zusammenspiel von ADO.NET mit XML wird gut berücksichtigt.
