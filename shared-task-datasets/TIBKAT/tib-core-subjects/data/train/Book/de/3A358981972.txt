Visual Basic .NET : objektorientiert programmieren von Anfang an
Mit Visual Basic .NET beschreibt der Autor die derzeit aktuellste Fassung dieser populären Programmiersprache, die neben C++ und Java eine große Verbreitung aufweist. Grundlegende Einführungen sollten daher in keiner Bibliothek fehlen. Der Autor hat bereits zu einer früheren Version dieser Sprache einen Titel vorgelegt (VB 5 in BA 2/98). Zwischenzeitlich erschien die noch immer genutzte Version Visual Basic 6. Die vorliegende Version wurde stark überarbeitet und auf das .NET-Framework abgestimmt. Das didaktisch gut aufbereitete Werk führt anhand von Beispielen Schritt für Schritt in die Programmiertechnik ein. Zusammenfassungen und Übungen (Lösungen im Anhang) erleichtern das Verständnis. Zum Thema existieren bereits eine ganze Reihe von Einsteigertiteln, so etwa der von Stefan Fellner (BA 9/03), der nach wie vor 1. Wahl ist. Ferner Peter Monadjemi (BA 4/03) und Jürgen Bayer (BA 11/02). Neben den beiden letztgenannten gerne empfohlen. (2)
