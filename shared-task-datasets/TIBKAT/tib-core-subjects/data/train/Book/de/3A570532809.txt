Java 6: Anwendungen programmieren : von der GUI-Programmierung bis zur Datenbank-Anbindung
Der Autor H. Balzert gilt als Java-Spezialist und ist u.a. bekannt durch Werke über Java 5 (ID 37/06; ID 33/05). Diese Werke sollten durchaus noch im Bestand verbleiben. Die aktuelle Version von Java ist Java 6. Der Autor hat daher zusammen mit seinem Co-Autor das vorliegende Buch herausgebracht, das auf seinen o.a. Büchern aufsetzt und in die neuen Features der aktuellen Version einführt. Das Buch ist somit kein typisches Einsteigerbuch wie z.B. B. Steppan (BA 3/07). Beim Leser werden Kenntnisse in der strukturellen, prozeduralen und objektorientierten Programmierung mit Java vorausgesetzt. Schwerpunkte liegen bei der Gestaltung und Programmierung grafischer Benutzeroberflächen und beim Arbeiten mit relationalen Datenbanken. Sehr gut berücksichtigt werden auch praktische Anwendungen. Hier findet der Leser drei umfangreiche Fallstudien mit den Themen Auftragsverwaltung, Wetterstation und dem Spiel Othello (intelligente Spiele). Die didaktische Gestaltung des Buches ist verlagstypisch und durch frühere Bücher des Autors bekannt. Zum Selbststudium ist das Buch gut einsetzbar. (2)
