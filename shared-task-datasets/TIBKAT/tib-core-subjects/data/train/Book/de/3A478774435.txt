Java lernen mit Eclipse 3 : [Einführung in die Java-Programmierung ; Komplettpaket: JDK 5 und Eclipse 3 auf CD ; Objektorientierung, GUIs, Threads und Datenbanken]
Eclipse ist ein Entwicklungstool zur Programmierung in verschiedenen Programmiersprachen, insbesondere in Java; Eclipse gilt heute als die führende Entwicklungsumgebung für Java-Applikationen. Der Autor hat Eclipse in einer Darstellung für Fortgeschrittene ausführlich beschrieben und seine Beispiele in Java programmiert (ID 44/04). Das vorliegende Buch hat eine andere Zielsetzung. Es wendet sich an Einsteiger und führt in das Programmieren mit der aktuellen Version von Java (Java 5) ein. Eclipse wird hierbei als unverzichtbares Werkzeug angesehen, welches gegenüber dem Arbeiten mit einem Editor eine Vielzahl von Vorteilen bietet. Das Buch setzt auf das Prinzip des "Learning by doing" und gibt oft gut kommentierte Schritt-für-Schritt-Anweisungen. Geschickter didaktischer Aufbau. Zu jedem Kapitel Zusammenfassungen, Verständnisfragen, Aufgaben und Webseiten mit weiterführenden Informationen. Gute Hinweise für die Entwicklung eines individuellen und sauberen Programmierstils. Schwerpunkte liegen bei der objektorientierten Programmierung und beim Arbeiten mit Datenbanken. (2)
