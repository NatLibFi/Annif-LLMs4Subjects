OS X Lion : das umfassende Handbuch ; [inkl. OS X Server ; Grundlagen, Anwendung, Praxiswissen ; mit Boot Camp, Time Machine, Mission Control, iCloud, App Store u.v.m. , inkl., Automator, Terminal, AppleScript & Netzwerke ; aktuell zu Version 10.7]
OS X 10.7 Lion ist das seit Oktober 2011 verfügbare aktuelle Betriebssystem von Apple. Einführungen für Einsteiger sind u.a. A. Ochsenkühn (ID-A 36/11) oder D. Mandl (ID-B 50/11), die auch gut auf Änderungen gegenüber der Vorgängerversion eingehen. Der Autor des vorliegenden Buchs ist ausgewiesen als Mac-Experte. Er legt hier ein umfangreiches Nachschlagewerk vor, das auch Themen wie iCloud und den Einsatz von Windows und Linux auf dem Mac behandelt. Der Autor setzt beim Leser Grundkenntnisse über das Arbeiten auf dem Mac voraus und verzichtet auf die Erläuterung von Routinearbeiten. Am Anfang steht eine kurze Zusammenfassung der neuen Features von OS X Lion; er weist auch darauf hin, welche veralteten Technologien nicht mehr unterstützt werden. Schwerpunkte liegen bei der Installation und Konfiguration von OS X Lion und des Lion Servers, beim Arbeiten im Netzwerk, bei der Automatisierung von Arbeitsabläufen (Automator, Arbeiten mit Apple Script) und beim Troubleshooting. Zusammenstellung nützlicher Tools, die OS X Lion gut ergänzen. Viele Tipps und Trick, zahlreiche Abbildungen und Beispiele. (2)
