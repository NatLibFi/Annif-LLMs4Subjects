Spring im Einsatz
Mit der Version 5.0 bringt Spring eine Reihe von Neuerungen mit sich, allen voran die Unterstützung von Java 9. Ausreichend Material für Craig Walls, um eine neue, gründlich überarbeitete Auflage seines Bestsellers „Spring im Einsatz“ vorzulegen. In dieser umfassenden Einführung finden Sie alles Wissenswerte über Spring 5.0. Sie steigen mit den grundlegenden Konzepten von Spring ein und lernen dann das Framework rasch aktiv kennen. Das Buch bietet eine vergnügliche Lektüre und gleichzeitig wertvolle Informationen, auf die Spring-Entwickler nicht verzichten können.
