Informatik kompakt : eine grundlegende Einführung mit Java
Rezension: Die Autoren geben einen Einstieg in die Informatik und wenden sich an Studierende. Die Informatik wird als Wissenschaft der Abstraktion herausgestellt. Vermittelt werden allgemeine Methoden zur Lösung praktischer Probleme. So wird die Bedeutung der Informatik für die Berufspraxis gut herausgestellt. Schwerpunkte liegen bei der Modellierung, abstrakten Datentypen, Algorithmen sowie nebenläufiger und verteilter Programmierung. Grundlegende Konzepte werden mittels der objektorientierten Sprache Java realisiert. Zusammenhänge zwischen den großen Teilgebieten der Informatik werden deutlich herausgearbeitet. Klar hervorgehobene Definitionen und tabellarische Übersichten. Die Autoren legen großen Wert auf eine Leistungskontrolle. So endet jedes Kapitel mit einem Abschnitt "Was wissen Sie jetzt?". Das Buch eignet sich gut zur Prüfungsvorbereitung. Der kompakte Index enthält nur prüfungsrelevante Begriffe. Wer zu jedem Stichwort die Definition und die Verwendung angeben kann, hat eigentlich genug gelernt. Umfangreiche Programmbeispiele im Internet. Interessante Alternative etwa zu W. Küchlin (ID 6/05).
