Active Directory : [Handbuch für Administratoren ; Profiwissen zu Microsofts Verzeichnisdienst ; kompakt, praxisnah und projekterprobt ; durchgängiges Referenzbeispiel]
Das Active Directory (AD) ist ein Verzeichnisdienst von Microsoft. Es ermöglicht als zentrales Element von Windows 2000/2003 die einheitliche objektorientierte Verwaltung von Dateien, Applikationen, Benutzern, Ausgabegeräten und die Netzwerksicherheit. Eine umfassende, gute Einführung ins Active Directory, dem Verzeichnisdienst von Microsoft. Die Erklärungen sind logisch und nachvollziehbar. Der Autor erklärt das Thema aus der Praxis heraus und kann es sehr gut veranschaulichen. Der Leser wird dadurch gut in die komplexe Materie von Active Directory eingeführt. Anhand eines imaginären Unternehmens spielt der Autor viele Szenarien durch. Auch die Aufzählung und Beschreibung gängiger AD-Supporttools gefällt gut. Ein Stichwortverzeichnis und ein sehr ausführliches Glossar im Anhang. Dadurch eignet es sich auch als Nachschlagewerk zu immer wiederkehrenden Fragen des Verzeichnisdienstes. Keine CD-Beilage. Sehr begrenzte Zielgruppe, nur für Systemverwalter oder Administratoren. Nach kompletten Kompendien zu Windows Server, die sich natürlich auch mit dem AD befassen. (3)
