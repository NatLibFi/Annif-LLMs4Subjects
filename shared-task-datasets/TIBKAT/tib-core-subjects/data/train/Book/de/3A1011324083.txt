Java – kurz & gut
In der eingeführten "kurz & gut"-Reihe des O'Reilly-Verlags liegt ein weiteres schmales Referenzwerk zur Programmiersprache Java vor. Es enthält eine Auswahl der wichtigsten Begrifflichkeiten und eignet sich gut zum raschen Nachschlagen. Zuletzt angezeigt wurde die 2. Auflage, abgestimmt auf die Version Java 8. Auch diese 3. Auflage weist die gewohnte Dreiteilung in die Bereiche Sprache, Plattform und Anhänge auf. Als Neuerung hinzugekommen ist die Berücksichtigung der Version Java 9. Diese beinhaltet zwei neue Features: die Java-Shell (JShell) und das Java Module System. Letztere erlaubt eine Modularisierung des Quellcodes. Für beide Themen gibt es jeweils ein zusätzliches Kapitel. Die übrigen Kapitel blieben, bis auf drei kleinere Ergänzungen, weitestgehend unverändert. Der Umfang des Buchs hat um gut 40 Seiten zugelegt. Empfiehlt sich als Ergänzung zu Handbüchern (C. Ullenboom) und Einführungen (M. Kofler)
