PIC-Microcontroller-Programmierung
Bücher zum Einsatz von Mikrocontrollern z.B. zum Messen, Steuern und Regeln sind in großer Zahl im Franzis-Verlag erschienen (zuletzt W. Trampert: ID 44/04; D. Kohtz: ID 4/03). Diese Bücher wenden sich aber überwiegend an engagierte Praktiker und setzen ein erhebliches Vorwissen voraus. Das vorliegende Buch bietet hierzu eine interessante Alternative. Es wendet sich bewusst an Einsteiger und setzt bei der Programmierung auf eine Variante von Basic. Verdeutlicht werden soll insbesondere das Zusammenspiel von Hardware und Software. Den Beispielen liegt die allgemein erhältliche Platine iL-TROLL-TB (Bezugsquellen im Anhang) sowie der vorprogrammierte Baustein iL-TROLL zugrunde.Weitere Software kann aus dem Internet heruntergeladen werden. Die einfachen Beispiele können leicht für eigene Zwecke variiert werden. Ebenso ist eine einfache Übertragung auf beliebige Mikrocontroller möglich. Der Aufbau der Beispiele wird jeweils von mehreren Funktionstest begleitet, was ein zusätzliches Sicherheitsgefühl, vermittelt. - Die Autoren erklären sich bereit, jederzeit Leseranfragen per E-Mail zu beantworten.
