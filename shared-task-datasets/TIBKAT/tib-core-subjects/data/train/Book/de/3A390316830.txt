Java 2
Java ist eine objektorientierte, weitestgehend plattformunabhängige Sprache, die für das Internet von großer Bedeutung ist. JDK (Java Development Kit) 5.0 ist die aktuelle Version der Java-Entwicklungsumgebung. Hierauf basiert die vorliegende, stark überarbeitete Auflage eines Standardwerkes zu Java (frühere Auflagen wurden hier nicht besprochen). Der Sachtitel führt etwas in die Irre, da die Namensgebung für die aktuelle Version von Java nicht ganz einheitlich ist. Der Autor behandelt die gleiche Thematik, wie sie unter dem Sachtitel "Java 5" von D. Louis (BA 3/05) oder F. Esser (BA 1/05) behandelt wird. Der Autor setzt einen Schwerpunkt bei den aktuellen Neuerungen wie z.B. Generics oder Auto-Boxing. Ausführlich wird auch die Grafikprogrammierung unter Java, z.B. Swing zur Programmierung grafischer Benutzeroberflächen, erläutert. Klassische Teile von Java wie z.B. AWT (Abstract Windowing Toolkit) sind auf CD-ROM ausgelagert. Java-Applets werden nur kurz gestreift. Didaktisch gut gestaltet, viele Tipps und Tricks, viele wichtige Internet-Adressen. Insbesondere für Einsteiger vor den o.a. Titeln. (2)
