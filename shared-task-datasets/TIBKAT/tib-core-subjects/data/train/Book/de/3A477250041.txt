Objektorientierte Programmierung mit Java 5 : mit CD-ROM und e-learning-Online-Kurs
Das Buch versteht sich als Einführung in die objektorientierte Programmierung und zeigt den Weg von der objektorientierten Analyse über den objektorientierten Entwurf hin zum objektorientierten Programm. Für die Modellierung wird hierbei UML (Unified Modeling Language) 2 verwendet. Die Beispiele werden in Java 5 gegeben. Das Buch orientiert sich nicht an der Syntax von Java 5. Gleichwohl werden alle wichtigeren Sprachkonzepte von Java 5 ausführlich dargestellt. Das Buch ist so eine interessante Alternative zu Werken über Java 5 wie B. Steppan (BA 3/05 - Einführung) oder D. Louis (BA 3/05 - Kompendium). Das Buch setzt keine besonderen Vorkenntnisse voraus und ist als Begleitunterlage zu Vorlesungen konzipiert. Aufgrund der vielen Beispiele und Aufgaben (oft mit Lösungen auf der Begleit-CD-ROM) kann es auch gut zum Selbststudium eingesetzt werden. Auf CD-ROM viele hilfreiche Programme (Entwicklungsumgebung, Tools, ...). Didaktisch sehr gut gestaltet; viele grafische Hervorhebungen und viele Tabellen. Die Aufgaben sind unterteilt in Muss-, Kann- und Klausuraufgaben - mit Zeitvorgaben für die Lösung. (2)
