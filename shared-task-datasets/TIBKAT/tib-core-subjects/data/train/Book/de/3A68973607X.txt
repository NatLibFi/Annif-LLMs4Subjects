jQuery : vom Novizen zum Ninja ; [Profi-Wissen füe alle jQuery-User ; das JavaScript-Framework professionell einsetzen ; interaktive Elemente in Webseiten integrieren ; eindrucksvolle Animationen mit wenigen Zeilen Code erstellen]
jQuery ist eine JavaScript-Bibliothek, die eine wesentliche Hilfe bei der Gestaltung von Webseiten bietet. jQuery kann plattformunabhängig eingesetzt werden. Ein umfangreiches Handbuch zum Thema ist F. Bongers (ID-B 43/11), eine praxisorientierte Darstellung mit vielen Beispielen gibt das "jQuery-Kochbuch" (ID-G 33/10), Einsteiger seien auf D. Mies (zuletzt BA 5/10) hingewiesen. Die Autoren des vorliegenden Buches wenden sich in ihrer locker geschriebenen, gut nachvollziehbaren Darstellung an fortgeschrittene Einsteiger, die über Grundkenntnisse in HTML und CSS verfügen sollten. Kenntnisse in JavaScript erleichtern die Lektüre. Das Buch beginnt mit den Grundelementen von jQuery, kommt dann aber sehr schnell zur Auswahl von Zielelementen (Selektion) auf einer Webseite und deren Animierung. Gut berücksichtigt werden hierbei Bildergalerien und Diashows. Weitere Schwerpunkte liegen bei grafischen Benutzeroberflächen, Formularen, Kontrollelementen, Listen und Tabellen. Das Buch schliesst mit jQuery Mobile, einer Einsatzmöglichkeit von jQuery für mobile Geräte. Viele Hinweise auf typische Stolpersteine. (2)
