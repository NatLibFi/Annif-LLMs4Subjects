Der Einfluss von Cloud Computing auf IT-Dienstleister : Eine fallstudienbasierte Untersuchung kritischer Einflussgrößen
Cloud-Computing -- Auswirkungen auf IT-Dienstleister -- Geschäftsmodelle -- Fallstudie -- Unternehmensstrategie.  .
Obwohl Cloud Computing noch am Beginn seiner Verbreitung steht, prophezeien Marktstudien ein außerordentliches Wachstum. Da Cloud Computing ein Substitut zum herkömmlichen Outsourcing darstellt, wird mit Auswirkungen auf den Markt für traditionelle IT-Dienstleistungen gerechnet. Raimund Matros analysiert IT-Dienstleister, die bereits Cloud-Computing-Dienste anbieten oder eine Einführung planen. Nach der Entwicklung einer Typologie und der Durchführung einer Marktstudie werden relevante Wirkungseinflüsse theoretisch abgeleitet und empirisch im Rahmen von Fallstudien untersucht. Das Analyseergebnis liefert darüber hinaus einen Ausblick auf zukünftige Differenzierungspotenziale für IT-Dienstleister.
