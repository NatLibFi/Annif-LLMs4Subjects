MySQL Kochbuch : [behandelt MySQL 4.0]
MySQL 4.0 ist ein leistungsstarkes Datenbanksystem, das einen verhältnismäßig geringen Ressourcenbedarf hat und daher weitverbreitet ist. Umfassende Darstellungen zu MySQL wurden von M. Throll (BA 11/03) und M. Barth (BA 6/02) vorgelegt. Das "Kochbuch" hier ergänzt diese Darstellungen ideal, indem es eine übersichtliche Sammlung von Problemstellungen mit Lösungen bringt. Die angebotenen "Rezepte" sind Bausteine, die leicht in eigene Anwendungen eingebaut werden können. Genau erklärt wird auch, wie und warum die angebotenen Rezepte funktionieren, sodass die erlernten Techniken auf ähnliche Situationen übertragen werden können. Die Rezepte verwenden verschiedene Programmierschnittstellen, z.B. Java, Perl, Python und PHP. Schwerpunkte liegen u.a. beim Import und Export von Daten, beim Zugriff auf mehrere Tabellen und beim Einsatz von MySQL im Web (Einbindung von Abfrageergebnissen in Webseiten, Verabeitung von Webeingaben mit MySQL, MySQL-basierte Verwaltung von Web-Sessions). Viel theoretisches Hintergrundwissen in Anhängen. (3) (Klaus Barckow)
