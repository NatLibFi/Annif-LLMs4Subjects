CSS : die praktische Referenz
In der bekannten Reihe wird erstmals ein Titel zum Thema Cascading Style Sheets (CSS) vorgelegt. Diese Formatierungssprache wird ergänzend zu HTML und anderen Auszeichnungssprachen eingesetzt. Aufmachung und inhaltlicher Aufbau sind reihentypisch, hier beginnend mit einem Index der CSS Befehle, gefolgt von einem kurzen Einführungsteil. Im Mittelpunkt des Buches stehen die Grundlagen von CSS sowie, in den nachfolgenden 8 Kapiteln, die Funktionen und Befehle. Durch CSS substituierte HTML-Befehle werden abschließend im Anhang nachgewiesen. Ein Register für die Stichwortsuche beendet dieses Nachschlagewerk. Um aktuellen Entwicklungen Rechnung zu tragen, wird auf Unterschiede in den Versionen hingewiesen, dazu gehören auch erkennbare Veränderungen durch die noch im Entwurfsverfahren befindliche Version CSS 3. Neben Einführungen (Florence Maurice: BA 4/06; Helma Spona: BA 4/06) und vor ausführlicheren Darstellungen (Eric Meyer: BA 3/05 und ID 48/05) gut einsetzbar. (2)
