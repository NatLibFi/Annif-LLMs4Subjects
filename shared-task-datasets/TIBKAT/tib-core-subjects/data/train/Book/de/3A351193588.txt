PHP [Version 4]
Im Web werden heute zunehmend individuell aufbereitete Informationen angeboten. In der Qualifikationseinheit (QE) PHP lernen Sie einen Vertreter der Skriptsprachen kennen, die für die Entwicklung dynamischer Webseiten konzipiert wurden. Um eine arbeitsfähige Umgebung für PHP herzustellen, werden die grundlegenden Schritte der Installation und Konfiguration von Webservern erläutert. Sie werden anhand von praktischen Beispielen mit der Syntax von PHP Version 4 vertraut gemacht. Die praktischen Übungen sind so angelegt, dass Sie schrittweise ein einfaches, serverseitiges PHP-Shopsystem programmieren.
