Microsoft Expression Web 4 : das offizielle Trainingsbuch ; [machen Sie sich fit für Expression Web 4]
Die bekannte und kompetente Autorin hat schon früher ein Buch zu Expression Web verfasst, das von Rainer Burghausen für Neueinsteiger empfohlen wurde (ID 38/07). Inzwischen ist der Editor für professionelle Webseiten seit 2010 bei Version 4 angelangt. Noch immer wird von Microsoft herausgestellt, dass standardkonforme Webseiten erzeugt werden, eigentlich eine Selbstverständlichkeit, nur eben nicht für MS, wo man sich früher darum nicht kümmerte. Als neu (aber schon in Version 3 vorhanden) wird von Frau Spona Super Preview angegeben, eine nützliche Funktion, um die gestaltete Webseite in verschiedenen Browsern anzusehen (wenn diese installiert wurden oder man online testen kann). Wichtig für die Anschaffung des Buchs ist auch, dass Expression Web inzwischen Konkurrenz durch gute Gratis-Produkte bekommen hat, somit eher professionelle Webdesigner anspricht. Neulinge auf diesem Gebiet werden das gut strukturierte, mit vielen Schwarz-Weiss-Screenshots illustrierte Buch schätzen, das als "offizielles Trainingsbuch" den früheren Band von Chris Leeds: "Microsoft Expression Web 2" (ID-B 8/09) verdrängt.
