JavaFX : dynamische und interaktive Java-Applikationen mit JavaFX
Rezension: JavaFX ist eine neue Entwicklung von Sun, die auf der Java-Plattform basiert und speziell für die Entwicklung von Rich Internet Applications entworfen wurde. Rich Internet Applications sind sehr bedienungsfreundliche Anwendungen (Drag-&-Drop, Bedienung auch über Tastaturkürzel), die Internet-Techniken benutzen und eine intuitive Benutzeroberfläche bieten. Das vorliegende Buch führt anhand einer Betaversion von JavaFX Script in die Programmierung von plattformneutralen Anwendungen ein. Es ist somit eine interessante Alternative zu Ch. Kazoun (ID 15/08), wo die Programmierung von Rich Internet Applications mit der Konkurrenz-Software Adobe Flex beschrieben wird. Angesprochen werden Einsteiger, bei denen Grundkenntnisse im Programmieren (in einer beliebigen Programmiersprache) vorausgesetzt werden. Praxisnahe Darstellung mit vielen Beispielen. Schwerpunkte liegen u.a. bei der Grafik-Ausgabe in JavaFX und bei grafischen Benutzeroberflächen/Swing-Applikationen. Sehr hilfreich sind auch zahlreiche Hinweise zur Weitergabe von JavaFX-Applikationen mit Java Web Start über das WWW. (3)
