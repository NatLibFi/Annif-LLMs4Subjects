CliXX Java 3D : der Grundkurs
Interaktive dynamische dreidimensionale Darstellungen gewinnen im Internet, insbesondere bei E-Commerce, eine immer größere Bedeutung. Die Autorinnen stellen Java 3D als plattformunabhängige Programmierschnittstelle (API) und Erweiterung von Java vor. Sie führen in die Entwicklung interaktiver 3D-Grafikanwendungen und Animationen mittels Java 3D ein. Beim Leser werden Erfahrungen mit Java und Grundkenntnisse in 3D-Grafik vorausgesetzt. Die Ausführungen basieren auf den Betriebssystemen Windows und Linux. Die Darstellung ist als Multimedia-Kurs konzipiert; die CD-ROM stellt das Hauptwerk dar, das Buch gibt Erläuterungen. Aufgebaut ist der Kurs in voneinander unabhängigen Lehrmodulen, die sich auf 3D-Applets (ebenfalls auf CD-ROM) stützen. Der Quellcode dieses Applets kann leicht modifiziert und an eigene Bedürfnisse angepasst werden. Schwerpunkte liegen bei der Installation und Konfiguration von Java 3D und bei der Konzeption und dem Aufbau von Java 3D-Programmen. Gut berücksichtigt werden auch die Themenkreise Beleuchtung von Szenen und Hintergrundgestaltung. Aufgaben mit Musterlösungen. (2) (Klaus Barckow)
