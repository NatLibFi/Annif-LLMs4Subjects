Java : der Sprachkurs für Einsteiger und Individualisten
Java ist eine weitverbreitete objektorientierte Programmiersprache, die sich durch ihre Plattformunabhängigkeit auszeichnet. Eine gute Einführung für Einsteiger gibt R. Steyer (ID-A 22/12). Hingewiesen sei ferner auf B. Steppan (ID-A 50/11), wo auch Java-Tools vorgestellt werden. Das vorliegende Buch wendet sich ebenfalls an Einsteiger, geht aber von geringeren Voraussetzungen aus. Es stützt sich auf viele Übungen und Beispiele (mit Code-Listings im Internet), die teilweise etwas drastisch ausfallen (z.B. Fälschung von Lottozahlen). Didaktisch ist das Buch gut aufgebaut, sehr hilfreich sind die grafisch hervorgehobenen Spicker, Hinweise und Warnungen. Auch zum Selbststudium ist das Buch gut geeignet. Der Autor beginnt mit einem Crashkurs über die Grundlagen von Java und der objektorientierten Programmierung (Methoden, Objekte, Arrays, Strings, ...). Dann steigt er in die Programmierung ein. Gut berücksichtigt werden hierbei das Arbeiten mit Datenbanken, die Programmierung grafischer Benutzeroberflächen und die Nebenläufigkeit. Auch Besonderheiten beim Erstellen von Android-Apps werden betont. (2) (Klaus Barckow)
