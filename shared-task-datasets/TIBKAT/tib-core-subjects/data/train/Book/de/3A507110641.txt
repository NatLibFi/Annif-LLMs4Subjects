Das Eclipse-Codebuch : 182 Tipps, Tricks und Lösungen für Eclipse-spezifische Probleme
Eclipse ist eine Entwicklungsplattform, die von IBM entwickelt worden ist und die durch eine spezielle Plugin-Architektur sehr anpassungsfähig ist. Eclipse ist insbesondere für das Programmieren von Java-Applikationen von Bedeutung. Der Autor ist bekannt durch Einführungen in das Arbeiten mit Eclipse (zuletzt ID 49/05 - Eclipse 3.1 und Java 5). Das vorliegende Buch ist hierzu eine gute Ergänzung. Es bietet in Form von ca. 180 Templates ausführlich erläuterte Lösungen für häufig auftretende Programmierungsfragen an. Die Templates können wie Bausteine bei der eigenen Programmierung eingesetzt und für eigene Bedürfnisse leicht abgewandelt werden. Hinzuweisen ist darauf, dass der Autor nicht nur den Einsatz von Eclipse als reines Entwicklungswerkzeug behandelt. Auch die Benutzung der Eclipse-Plattform als Programmrahmen und Ablaufumgebung für eigene Plugins und so genannte RCP-Anwendungen (RCP = Rich Client Platform) wird ausführlich diskutiert. Gute Kenntnisse in Java 5 werden beim Leser vorausgesetzt. Auf Begleit-CD sämtliche Templates des Buches sowie Eclipse 3.1 für Windows, Linux und Mac OS X. (3)
