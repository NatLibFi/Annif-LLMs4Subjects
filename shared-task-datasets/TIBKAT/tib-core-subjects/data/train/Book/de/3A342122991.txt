MySQL für Dummies : [die Welt der Datenbanken! ; relationale Datenbanken mit MySQL entwickeln ; MySQL als Datenbankserver: eine super Verbindung zu Access mit ODBC ; auf der CD: MySQL, PHP und Apache]
SQL ist eine standardisierte Datenbanksprache, die in unterschiedlicher Vollständigkeit in Datenbankprogrammen implementiert worden ist. MySQL ist ein kostenfrei erhältliches Datenbankprogramm, das durch eine recht spartanische Oberfläche nur wenig Systemressourcen verbraucht und sehr schnell ist. MySQL wird zusammen mit der Skriptsprache PHP und dem Webserver Apache bei interaktiven Webauftritten eingesetzt. Der Autor führt in flotter, aber exakter Darstellung in das Arbeiten mit MySQL Version 3.12.43 ein. Als Werkzeuge für Webserver und Datenbankanwendungen im Web nutzt er PHP 4.0, Perl 5.0 und Apache 1.3x. Gut berücksichtigt wird die Nutzung von MySQL als Datenbankserver über komfortablen grafischen Oberflächen wie z.B. Microsoft Access. Reihentypische Aufmachung mit vielen Cartoons und einem Top-Ten-Teil. MySQL-Sprachreferenz im Anhang. Viele instruktive Beispiele. Auf CD-ROM alle benötigten Programme. Etwas anspruchsvoller als R. Däßler (BA 10/01). Parallel zu dem etwas älteren Werk von G. Leierer (BA 11/00). (2) (Klaus Barckow)
