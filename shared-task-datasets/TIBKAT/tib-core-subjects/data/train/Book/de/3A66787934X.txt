Erfolgreich Java 7 programmieren : [der grundlegende Einstieg in Java und die objektorientierte Programmierung ; von einfachen Java-Beispielen bis zu fortgeschrittenen Java-Applikationen ; inklusive der Behandlung wichtiger Tools im Java-Umfeld (JDK, Eclipse, NetBeans) und des Java-Standard-API]
Rezension: Java ist eine weitverbreitete objektorientierte Programmiersprache, die insbesondere von ihrer Plattformunabhängigkeit profitiert. Die aktuelle Version ist Java 7. Einführungen sind B. Burd (ID-A 19/12) oder B. Steppan (ID-A 50/11). Eine umfassende Darstellung gibt D. Louis (ID-B 4/12). Der Autor des vorliegenden Buchs ist Spezialist für Java und durch eine Vielzahl von Büchern, auch über Eclipse und JavaScript, bekannt. Er wendet sich hier an ambitionierte Einsteiger, die möglichst schon einmal (in einer beliebigen Sprache) programmiert haben sollten. Er stützt sich auf viele Beispiele und hebt gleich zu Anfang die Bedeutung der objektorientierten Programmierung (im Vergleich zur prozeduralen Programmierung) hervor. Auch Java-Entwicklungstools werden sehr früh berücksichtigt. Als Entwicklungsumgebungen dienen Eclipse und NetBeans. Schwerpunkte liegen u.a. bei Klassen, der Vererbung, Polymorphie und Generics. Aber auch die Erstellung grafischer Benutzeroberflächen, Multithreading und der Zugriff auf Datenbanken werden gut berücksichtigt. Öfter Hinweise auf weiterführende Internetquellen. (1)
