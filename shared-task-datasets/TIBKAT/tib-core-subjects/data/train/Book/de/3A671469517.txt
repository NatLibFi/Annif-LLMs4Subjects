Microsoft SQL Server 2012 - das Handbuch : [Insider-Wissen praxisnah und kompetent]
SQL Server 2012 ist die aktuelle Version eines oft eingesetzten relationalen Datenbanksystems von Microsoft. Ein umfassender Praxisratgeber für die Datenbankadministration ist W. Stanek (ID-B 34/12). Eher an Einsteiger wendet sich R. Panther (ID-B 31/12). Das vorliegende Buch richtet sich wieder an Entwickler und Administratoren. Es geht gut auf die neuen Cloud-Fähigkeiten und die Zusammenarbeit mit dem Microsoft Exchange Server 2010 und dem Microsoft SharePoint Server 2010 ein. Der Autor beginnt mit einer Zusammenfassung der neuen Features der aktuellen Version und mit Hinweisen zur Installation und zur Migration von älteren Versionen. Es folgen Ausführungen zum Anlegen und Verwalten von Datenbanken sowie zur Datensicherung und Wiederherstellung. Gut berücksichtigt werden auch die Systemüberwachung, Hochverfügbarkeit, Benutzerverwaltung und das Erstellen von Berichten. Gut erschlossen über ein tief gegliedertes Inhaltsverzeichnis und ein umfangreiches Stichwortverzeichnis (16 dicht bedruckte Seiten). Viele Praxistipps. Begleit-CD mit dem Inhalt als E-Book und vielen weiterführenden Links. (3)
