Java 7 : das Handbuch ; [Einführung in die Programmierung, Objektorientierung, GUI-Windows-Anwendungen, Drucken, Container und andere Utilities, Datenbanken, Threads, Lokalisierung, XML, Webservices, hunderte ausführbare Codebeispiele]
Wie zuletzt bei ihrem Titel der "Jetzt lerne ich ..."-Reihe zu Java 7 (ID-A 46/11) haben die beiden Autoren auch ihr bewährtes Handbuch für Einsteiger und Fortgeschrittene zu Java 6 (BA 7/07) nicht völlig neu bearbeitet, sondern die vorhandene Gliederung praktisch unverändert beibehalten und die Inhalte nur dort, wo notwendig, aktualisiert (z.B. neue Screenshots). Die nicht sehr üppigen Neuerungen von Java 7 werden nur kurz am Anfang mit Verweis auf die entsprechenden Kapitel aufgelistet. Ansonsten ist ein Kapitel zu Variablen neu aufgenommen, eine allgemeine Einführung in die Programmierung dafür entfallen. Wichtigstes Plus ist ein über 60 Seiten starkes Bonuskapitel zur Programmierung von Android-Apps, bei der Java eine wichtige Rolle spielt. Sollte den noch gut einsetzbaren Vorgänger überall ergänzen und bietet sich zusätzlich zu der oben genannten Ersteinführung oder der von B. Steppan (ID-A 50/11) an. G. Krüger (ID-B 50/11) und F. Jobst (ID-A 46/11) richten sich an einen Benutzerkreis mit mehr Vorkenntnissen. (2)
