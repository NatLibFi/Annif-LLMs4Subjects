Apps entwickeln mit Android Studio : Schritt für Schritt zur Android-App ; der perfekte Einstieg in die App-Entwicklung - GUI, Navigation, Kamera, Sensoren u.v.m.
Apps für mobile Android-Geräte gewinnen zunehmend an Bedeutung. Dies spiegelt sich auch in der besprochenen Literatur wider. Hinzuweisen ist hier u.a. auf B. Krämer (ID-B 24/14) oder auf U. Post (ID-G 32/14), der sich zwar auf die Programmierung von Spielen beschränkt, aber bereits die freie integrierte Entwicklungsumgebung Android Studio benutzt. Das vorliegende Videotraining beschränkt sich nicht auf Spiele, sondern gibt eine allgemeine Einführung und setzt beim Nutzer nur geringe Vorkenntnisse voraus; Grundkenntnisse in Java erleichtern aber das Arbeiten. Der Nutzer erhält zunächst einen kurzen Überblick über Entwicklungsumgebungen für Android-Programmierer und wird dann in die Installation von Android Studio und Java SDK eingeführt. Auch werden Java-Grundlagen noch einmal kurz zusammengefasst. Die Programmierung von Apps wird dann ausführlich anhand zweier Apps erklärt, einer sehr einfachen App für das Schäfchenzählen und einer komplexeren App "Quo vadis" mit Navigation. Gut berücksichtigt wird hier das Steuern von Android-Sensoren. Am Schluss nützliche Hinweise zur Vermarktung von Apps. (2)
