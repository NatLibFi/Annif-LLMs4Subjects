Einfach Webseiten gestalten mit Expression Web 2 : [so erstellen Sie Schritt für Schritt die eigene Website ; so gestalten Sie Ihre Website professionell ; so erreichen Sie mehr Traffic mit Ihrer Website]
Microsofts neuer Webeditor "Expression Web 2" gehört zu einer Produktfamilie, die alle Hilfsmittel anbietet, die man für eine eigene Website braucht. Microsofts früherer Webeditor "FrontPage" hatte zwar viele Funktionen, scherte sich aber wenig um Webstandards, sodass manche Webseiten nur im Internet Explorer richtig dargestellt wurden. Jetzt wird Code erzeugt, der dem Webstandard entspricht. Der erfahrene Internet-Autor Seimert legt hier eine Einführung in den neuen Editor vor. Der Band hat ein gutes Layout mit durchgehend farbigen Abbildungen, er ist für Einsteiger gedacht, die in Nebenbemerkungen z.B. auch erfahren, was ein HTML-Tag ist. Alle wichtigen Menüs und Funktionen werden in Schritt-für-Schritt-Anleitungen auf der Grundlage eines fiktiven Beispiels erläutert. Webprofis, die sich bereits intensiv mit dem Vorgänger FrontPage befasst haben, werden den umfangreichen (und teureren) Band "Microsoft Expression Web 2" von Helmut Vonhoegen (in dieser Nr.) vorziehen; für interessierte Laien und Neulinge aber ist Seimerts Buch ein guter Begleiter bei der Gestaltung ihrer Webseite.
