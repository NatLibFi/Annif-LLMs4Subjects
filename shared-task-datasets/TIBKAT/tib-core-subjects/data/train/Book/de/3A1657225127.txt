Realisierung der Einsparpotentiale bei elektrischen Energieverbrauchern
Optimierung der energieeffizienten Asynchronmaschine -- Visualisierung des Datenmodells Motordimensionierung -- Schnittstelle von IT und elektrischen Antrieben -- Visuelle Java Entwicklungsumgebung -- IT-Lösungen für energieeffiziente Optimierung des Asynchronmotors -- Drehzahl-Drehmoment-Kennlinie der Asynchronmaschine.
Das Buch stellt eine Projektmanagement-Anwendungsentwicklung bezüglich des energieeffizienten Asynchronmotors mit dem Open Source Java XDEV 4 Framework dar und bietet viele neue Möglichkeiten auf der Oberfläche. Das Buch fokussiert auf IT-Lösungen zu den Einsparpotentialen in Asynchronmaschinen. Mit Hilfe der Programmierung lassen Energieeffizienzsteigerungen erzielen. Die Energietechnik-Informatik ist somit ein Zukunftstrend. Der Inhalt Optimierung der energieeffizienten Asynchronmaschine Visualisierung des Datenmodells Motordimensionierung Schnittstelle von IT und elektrischen Antrieben Visuelle Java Entwicklungsumgebung IT-Lösungen für energieeffiziente Optimierung des Asynchronmotors Drehzahl-Drehmoment-Kennlinie der Asynchronmaschine Die Zielgruppen Informatiker, Wirtschaftsinformatiker und Ingenieure, die das Thema Energietechnik-Informatik in ihren Projekten anwenden möchten Elektroniker der Energietechnik sowie der Betriebstechnik, die Anwendungen für Energiemanagement mit Hilfe der IT entwickeln möchten Entwickler, die mit Hilfe von Java auf Datenbank MySQL zugreifen möchten Der Autor Eric Aristhide Nyamsi, M.Sc., ist Freiberufler für Engineering und Software-Entwicklung in Karlsruhe und entwickelt Anwendungen auf Basis von Java für die Schnittstelle von IT und Energie.
