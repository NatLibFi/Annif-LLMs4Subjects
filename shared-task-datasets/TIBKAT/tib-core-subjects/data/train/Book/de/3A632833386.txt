Einstieg in Java : das Praxis-Training ; [in 8 Stunden zum Java-Programmierer, inklusive Datenbanken, Servlets und Netzwerke mit Java]
Java ist eine weitverbreitete objektorientierte Programmiersprache. Eine gute praxisorientierte Einführung in das Programmieren mit Java ist das Buch von R. Schiedermeier (BA 11/10), das auf dem Entwicklungssystem Java SE JDK 6 basiert. Das vorliegende Videotraining ist eine ideale Alternative in Non-Print-Form. Der Trainer dieses Videokurses stützt sich ebenfalls auf SE JDK 6. Im Unterschied zu der o.a. Monografie werden im Videotraining aber auch die Programmierung grafischer Benutzeroberflächen mit Swing, der Zugriff auf Datenbanken, das Arbeiten in Netzwerken und das Programmieren von Servlets ausführlich behandelt; das Videotraining geht also weiter. Am Anfang stehen die Installation von Java und der Entwicklungsumgebung Eclipse. Ein sehr ausführlicher Schwerpunkt liegt bei der objektorientierten Programmierung. Gut berücksichtigt werden auch Exceptions und Assertions und das Java Collections Framework. Gute Navigation. Unterschiedliche Ansichtsmodi, bei denen der Mini-Player es optimal erlaubt, die Workshops des Trainers parallel zum laufenden Video selbst praktisch nachzuvollziehen. (2)
