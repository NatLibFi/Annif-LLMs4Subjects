Java ist eine Sprache : Java lesen, schreiben und ausführen - eine präzise und verständliche Einführung
Java ist eine Programmiersprache, die u.a. im Internet eine große Bedeutung hat. Die aktuelle Version ist Java 5. Eine umfassende Darstellung von Java 5 hat zuletzt G. Krüger (BA 3/05 gegeben). Eine aktuelle Einführung für Einsteiger ist B. Steppan (BA 3/05). Das vorliegende Buch ist ebenfalls eine Einführung, stark anwendungsorientiert, beruhend auf Vorlesungen an der TFH Berlin. Ziel des Autors ist, in das Arbeiten mit Java möglichst einfach und verständlich einzuführen. Er verwendet oft anschauliche Begriffe aus der Allgemeinsprache: eine Variable wird z.B. als Behälter für Werte erklärt. Dennoch bleibt der Autor aber stets präzise. Die didaktisch geschickte Darstellung stützt sich auf viele Beispiele, Schritt-für-Schritt-Anweisungen und Aufgaben mit Lösungsvorschlägen. Weiterführende Informationen werden im WWW bereitgestellt. Zusätzlich wird das Buch durch ein annotiertes Literaturverzeichnis ergänzt mit guten Bewertungen der dort aufgelisteten Literatur. Auch auf die neuen Features von Java 5, wie Aufzählungstypen (enums) und generische Elemente wird sehr verständlich eingegangen. (1)
