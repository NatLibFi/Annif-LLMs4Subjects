Intelligent control systems using computational intelligence techniques
The book is written at postgraduate level and comprises N chapters dealing with the following subjects: intelligent control systems; computational intelligence techniques; nonlinear system identification; nonlinear control; fuzzy systems; neural networks; multiobjective evolutionary computing solutions; adaptive local linear modelling; nonlinear dynamical systems; Gaussian process; nonlinear modelling; neuro-fuzzy model construction; nearly optimal control; reinforcement learning; multiagent control; and fault diagnosis
