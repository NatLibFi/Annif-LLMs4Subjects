Pro Spring security : [securing your enterprise and web Spring and Grails applications]
Spring security is a comprehensive authentication / authorization framework built on top of the Spring Framework dedicated to providing a full array of security services for Java applications in a developer-friendly and flexible way. It adheres to the well-established practices introduced by the Spring Framework attempting to address all the layers of security inside your application. In addition, it comes packed with an extensive array of configuration options that makes it very flexible and powerful
