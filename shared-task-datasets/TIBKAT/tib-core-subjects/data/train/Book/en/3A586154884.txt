Beginning OpenOffice 3 : from novice to professional ; [a complete guide to OpenOffice3 - learn to create documents, spreadsheets, databases, web sites, graphics, and more with complete, open source office suite]
This book is designed to guide you through the process of creating a variety of projects while exposing many different tools and options to make your work look more professional and creative and save you time and effort. OpenOffice 3 is an application program that can run on a range of different systems, such as Mac, Windows, Linux, and Solaris, and includes word processing, spreadsheets,  presentations, databases, and illustrations all built around a common interface. Learn to create documents, spreadsheets, databases, web sites, graphics, and more, with this open source office suite
