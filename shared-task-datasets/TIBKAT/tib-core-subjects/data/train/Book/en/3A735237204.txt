Expert MySQL : [apply mastery of MYSQL internals to high-availability, replication, customization, and beyond]
Expert MySQL provides you with in-depth knowledge of internals to help you unlock the full promise of this open-source relational database-management system. It shows you how to modify the code, create your own storage engine, build your own authentication plugin, and even add your own functions and commands to the SQL language. You'll learn how to : learn the inner workings of MySQL ; implement high-availability solutions and replications ; modify and debug the server code ; build your own pluggable stoarge engines and authentication modules ; apply new SQL functions and syntax ; build your own query optimizer
