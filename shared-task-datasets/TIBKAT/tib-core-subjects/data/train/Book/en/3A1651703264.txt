Using R for introductory statistics
R is a free software package that has seen a huge increase in users within the last five years and now is used increasingly in introductory classes. There are few books covering introductory statistics using R, and this book fills a gap as a true ...beginner... book. With emphasis on data analysis and practical examples Using R for Introductory Statistics encourages understanding rather than focusing on learning the underlying language.
