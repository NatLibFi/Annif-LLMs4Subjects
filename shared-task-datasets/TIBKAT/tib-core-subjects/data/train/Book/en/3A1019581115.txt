Introduction to computer graphics with OpenGL ES
Unity dominates the game engine industry and has become the engine of choice for game companies. This book achieves a balance between theory and practice by presenting the fundamentals of game graphics and showing how they are implemented into Unity. Theoretical or mathematical detail around real-time graphics are presented in a way that allows readers to quickly move on to practical aspects of game programming, especially those adopted in Unity. This book also presents C# script code on many topics, providing a hands-on look at Unity graphics. Industry professionals as well as students in upper-level courses in Computer Graphics and Game Programming will find this book of immense value--
