Experimentelle Untersuchung des Aktivitätskoeffizienten bei unendlicher Verdünnung in ausgewählten Lösungsmitteln und Lösungsmittelgemischen als Grundlage für die Synthese thermischer Trennprozesse
Die vorliegende Arbeit befasst sich mit der Untersuchung der Aktivitätskoeffizienten bei unendlicher Verdünnung im Hinblick auf die Erhöhung der Selektivität bei technisch realisierten Trennprozessen. Dabei wird eine systematische Untersuchung verschiedener Lösungsmittel mit unterschiedlich flüchtigen Komponenten sowohl mit der Gas-Flüssig-Chromatographie als auch mit der Dilutor-Technik durchgeführt. Bei den Messungen mittels der eingesetzten Methoden konzentriert sich die Untersuchung auf Lösungsmittel, für die bisher keine oder nur wenig Daten vorliegen. Daneben wird im zweiten Teil der Arbeit ein Programm für die Auswahl von Absorptionsmitteln vorgestellt, das in ein bereits bestehendes Programmpaket integriert wurde. Mit diesem Programm ist die computergestützte Suche nach potenziellen Absorptionsmitteln mit Hilfe von Faktendatenbanken (DDB) bzw. unter Verwendung der PSRK-Gruppenbeitragszustandsgleichung möglich. <dt.>
This thesis deals with the investigation of the activity coefficients at infinite dilution with a view to increasing selectivity for technically implemented separation processes. Thereby a systematically investigation of different solvents with several solutes has been carried out using gas-liquid chromatography and the dilutor technique. The measurements were concentrated on solvents for which so far no or only few experimental data are available. Furthermore, a software program for the selection of absorbents, which was integrated in an already existing software package, is presented in the second part of this thesis. With the help of this program a computer based search of potentially absorbents is possible using factual databanks (DDB) or the PSRK equation of state. <engl.>
