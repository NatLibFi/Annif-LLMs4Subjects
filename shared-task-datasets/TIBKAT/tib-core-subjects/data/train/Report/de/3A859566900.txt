SQL Server 2017 : der schnelle Einstieg
Für die neueste Ausgabe seines Handbuchs zum Datenbankmanagementsystem Microsoft SQL Server hat Konopasek eine durchgehende Überarbeitung des Inhalts vorgenommen (zuletzt "SQL Server 2014"). Auch wenn der Untertitel von einem Einstiegswerk spricht, so geht das Gebotene doch weit darüber hinaus. Diesem nunmehr auf über 1.000 Seiten (vormals 747 Seiten) angewachsenen Buch liegt nicht nur die Version 2017 zugrunde, sondern es wird auch die kurz zuvor veröffentlichte Version 2016 berücksichtigt. Eingeflossen sind zahlreiche Neuerungen und Verbesserungen, die sowohl den Kern von SQL Server betreffen, als auch die Rahmenprodukte (Analyse-, Service- und Reporting-Dienste). Auch gibt es jetzt eine Verfügbarkeit für das Betriebssystem Linux. Ein hinzugefügtes Kapitel widmet sich diesem Thema. Wohl um das Einsteigermerkmal zu unterstreichen, hat sich der Anteil der grau abgesetzten Hinweis- und Infoboxen deutlich erhöht. Hinzu kommen Codebeispiele und Praxistipps.
