Schrödinger programmiert C# : das etwas andere Fachbuch
Ein Fachbuch zur objektorientierten Programmiersprache C (Version 8.0) aus der bekannten Reihe "Schrödinger programmiert ..." von Rheinwerk, bei der mittels comicartigem Stil Ein- und Umsteiger in Programmiersprachen eingeführt werden. Der Autor vermittelt C 8 unter Verwendung der Entwicklungsumgebung Visual Studio von Microsoft und .Net Core. Zum besseren Verständnis sind die Ausführungen mit zahlreichen Übungen und Codebeispielen versehen. Im Vergleich zur 2. Auflage ist der Inhalt zwar aktualisiert, aber überwiegend seiten- und textidentisch. Das Kapitel zum Publizieren im Windows Store ist entnommen. Ergänzend zu Th. Theis: "Einstieg in C mit Visual Studio 2019" und vor A. Kühnel: "C 8 mit Visual Studio 2019 ". Die 1. Auflage sollte ausgetauscht werden - auch zur Erstanschaffung empfehlenswert
