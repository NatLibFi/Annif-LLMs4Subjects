Python programmieren lernen für Dummies
Die Programmiersprache Python verfügt über einen klaren, gut zu lernenden Code und kann auf verschiedenen Plattformen eingesetzt werden, weshalb sie zu den beliebtesten Sprachen gehört. Sie dient unter anderem als Skriptsprache für Anwendungen wie Gimp oder Scribus und auch für den Raspberry Pi. Der Autor kommt schnell auf den Punkt und erklärt Ein- und Umsteigern, wie man mit der kostenlosen Distribution "Anaconda" und der Entwicklungsumgebung "Jupyter Notebook" seine ersten Python-Programme schreiben kann. Auf die Vermittlung der Grundlagen folgen alltägliche Beispiele, bevor auf anspruchsvollere Aufgaben wie das Speichern von Daten in Dateien eingegangen wird. Die Vorauflage basierte noch auf Python 3.4.2, die vorliegende auf 3.6.3; inzwischen gibt es 3.7.1, aber das schränkt den Nutzen in keiner Weise ein. Es gibt eine Reihe Unterschiede zwischen den Auflagen, vor allem in der Kapitelstruktur, dennoch kann die 1. noch im Bestand verbleiben und/oder um die vorliegende ergänzt werden. Nach T. Theis ("Einstieg in Python")
