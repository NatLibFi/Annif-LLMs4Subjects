Das Einsteigerseminar objektorientierte Programmierung in Java
Bewährte Einführung in Java und die Grundprinzipien der objektorientierten Programmierung für den Ersteinstieg, die inzwischen Java 7 berücksichtigt (zuletzt angezeigt: 4. Auflage, BA 12/04; kann verbraucht werden, ältere aussondern). Erläutert recht knapp, aber hinreichend ausführlich das Prinzip Objektorientierung beim Erstellen erster Programme unter Java. Geht sehr praxisgerecht anhand von realistischen Beispielanwendungen vor (z.B. Programmieren eines Ausleihprogramms für eine Schulbibliothek), wobei theoretische Abschnitte durch Beispiele und Übungen jeweils sinnvoll ergänzt werden (Lösungsteil am Ende des Buches). Kann gut neben dem Java-Übungsbuch des selben Autors (BA 5/09) stehen. Für einen breiteren Nutzerkreis als H. Balzert (BA 5/10) oder M. Block (BA 4/10), die vor allem Informatiker im Grundstudium im Blick haben. (2) (LK/M: Zimmermann)
Einführung in die Grundlagen der Programmierung mit Java und OOP: Installation, Sprachstrukturen, Dateiverarbeitung und grafische Oberflächen, Fehlerbehandlung; mit Praxisbeispielen. (LK/M: Zimmermann)
