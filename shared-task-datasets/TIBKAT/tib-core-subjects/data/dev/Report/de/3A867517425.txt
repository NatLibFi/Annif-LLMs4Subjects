Linux Mint 18 : Praxiswissen für Ein- und Umsteiger
Linux Mint ist eine Linux-Distribution, die auf Linux Ubuntu beruht und sehr gut auch Umsteiger von Windows anspricht. Das vorliegende Buch von Troche ist der Version Linux Mint 18 (Sarah) gewidmet, die auch auf CD-ROM beigefügt ist. Der Autor legt einen Schwerpunkt bei Umsteigern von Windows. Er beginnt mit einem Hinweis, für wen sich Linux nicht eignet (grosse Spiele) und stellt dann die verschiedenen Versionen von Linux Mint mit ihren Möglichkeiten vor. Weitere Schwerpunkte liegen dann bei verschiedenen Installationen (z.B. paralleler Betrieb von Linux Mint und Windows), bei Sicherheitsfragen, bei der Bürosoftware LibreOffice und bei Linux Mint in der Cloud. Sehr hilfreich ist das Schlusskapitel über die Einrichtung eines virtuellen persönlichen Netzwerks (VPN), das es ermöglicht, von jedem öffentlichen Hotspot eine sichere, verschlüsselte Verbindung zum heimischen Router zu erstellen
