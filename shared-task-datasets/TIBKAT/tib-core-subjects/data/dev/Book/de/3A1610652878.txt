Die eigene Website : 12 leichte Workshops für Einsteiger
Von Jacobsen gibt es bereits einen recht erfolgreichen Band zum Thema: "Website-Konzeption: Erfolgreiche Websites planen, umsetzen und betreiben" (6. Auflage = 1. Neuauflage 2011 lag nicht vor; 5. Auflage: ID-A 34/09). Jetzt hat er mit einem Koautor einen anderen Weg beschritten. "Die erste Website in 3 Minuten" verspricht der 1. von 12 Workshops. Dabei soll es aber nicht bleiben. Folgekapitel führen in ein wenig HTML und CSS ein und verwenden dann den kostenlos ladbaren "Seamonkey Composer". Schliesslich wird dann eine weitere "Website in 3 Tagen" erstellt, mehr CSS gelehrt und wichtige Aspekte einer Website wie Benutzerfreundlichkeit oder Optimierung für Suchmaschinen beschrieben. Sogar die verschiedenen Versionen der gängigen Browser werden berücksichtigt, die gleichen Code ja oft verschieden darstellen. Das alles wird mit vielen grosszügigen farbigen Abbildungen und Screenshots illustriert. Der gut lesbare quadratische Band ist auch praktisch: Er bleibt auch aufgeschlagen liegen, wenn er das soll. Er wird dankbare Leser finden. (2)
