SQL Server 2008 - der schnelle Einstieg : Abfragen, Transact-SQL, Entwicklung und Verwaltung ; [für Datenbankentwickler und -administratoren]
Rezension: SQL Server ist ein relationales Datenbankmanagementsystem von Microsoft, das jetzt in der aktuellen Version 2008 vorliegt. Eingesetzt wird es vorwiegend im kommerziellen Unternehmensbereich. Eine Darstellung auf der Basis einer Beta-Version hat bereits R. Dröge (ID 21/08) gegeben. Der Autor des vorliegenden Buches ist u.a. bekannt durch das Video-Training "SQL Server 2005-Programmierung" (ID 3/08). Er wendet sich auch hier nicht an Leser mit großen Vorkenntnissen. Angesprochen werden Einsteiger und Umsteiger von älteren Versionen wie auch von anderen Datenbanksystemen wie z.B. Access. Die neuen Features der aktuellen Version werden gut herausgestellt. Schwerpunkte liegen bei der Programmiersprache Transact-SQL und der grafischen Benutzeroberfläche SQL Server Management Studio. Gut berücksichtigt werden auch die .NET-Integration, Benutzerverwaltung und Berechtigungsvergabe sowie die Sicherung von Datenbanken und ihre eventuelle Wiederherstellung. Viele instruktive Abbildungen. Eine große Beispieldatenbank aus der Warenwirtschaft, die schrittweise aufgebaut wird. Auch zum Selbststudium geeignet. (2)
