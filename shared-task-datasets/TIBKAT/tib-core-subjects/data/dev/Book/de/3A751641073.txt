Praxiswissen Responsive Webdesign : [reaktionsfähige Websites für alle Geräte ; Strategien, Konzeption, Umsetzung ; mit vielen Beispielen aus der Praxis]
Der Autor richtet sich an Webdesigner und -entwickler, die über Grundkenntnisse in HTML und CSS verfügen und sich nun mit Responsive Webdesign vertraut machen wollen. Behandelt werden sowohl technische Aspekte als auch konzeptionelle Fragen, die dieser Designansatz mit sich bringt. (Christian Eidloth)
Responsive Webdesign heißt Webdesign, das auf das benutzte Endgerät, vulgo Smartphone oder Tablet, reagiert und das Layout dementsprechend anpasst. Und angesichts der rasant steigenden Zugriffe von mobilen Geräten ist es das Thema der Stunde. Das Buch ist weiter gefasst als vergleichbare Titel (z.B. P. Müller: ID-A 41/13), die den Fokus auf die Möglichkeiten mit HTML 5 und CSS 3 richten. Es beschreibt auf einer grundsätzlicheren Ebene Strategien und Konzeptionen, wie Webseiten für alle Geräte gestaltet und programmiert werden müssen. Es beginnt - eher traditionell - mit Fluid Layouts, Media Queries und Responsive Media und kommt dann zu Grundsätzlicherem wie Responsive Workflow, Responsive Content und Responsive User Experience. Eingestreut sind auch Studien und Erfahrungsberichte anerkannter Response-Experten, die Einblicke in die Praxis ermöglichen. Das Buch ist im Original auf Englisch; die Übersetzung ist - wie so oft - mitunter ungelenk , vergleichsweise aber noch im Rahmen. Für diejenigen, die Responsive Design wirklich als neues Paradigma begreifen, ein sehr wertvolles Buch. (1) (Christian Eidloth)
