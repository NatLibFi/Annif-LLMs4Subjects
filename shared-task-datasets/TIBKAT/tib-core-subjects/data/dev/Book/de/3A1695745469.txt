Windows Server für Dummies
Windows Server ist ein Serverbetriebssystem von Microsoft, basierend auf Windows 10; die Bedeutung für die moderne EDV liegt in der Zusicherung von Microsoft, dass die Belieferung mit Sicherheitsupdates bis zum Januar 2029 garantiert ist. Literatur zu Windows Servern ist hier nur selten besprochen worden, zu erwähnen sind u.a. Carlo Westbrook: "Windows Server 2019", Thomas Joos: "Microsoft Windows Server 2019", Jörg Schieb: "Windows Server 2019"  und Peter Kloep: "Windows Server 2019". Vorliegender Titel zeigt, wie man mit Windows Server gut arbeiten kann und legt Schwerpunkte bei GUI (grafische Benutzeroberflächen), Konsole und PowerShell. Ausserdem erfahren die Leser*innen, wie ein Active Directory eingerichtet wird, was man mit Gruppenrichtlinien machen kann, wie man eine Domäne erstellt, wie man mit Hyper-V virtualisieren kann, und wie man Daten sichern und Prozesse überwachen kann. - Das Buch hier setzt keine besonderen Vorkenntnisse voraus und eignet sich daher auch für Einsteiger*innen
