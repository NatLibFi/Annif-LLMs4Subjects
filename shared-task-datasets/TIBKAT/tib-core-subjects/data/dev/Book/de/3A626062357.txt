Grundkurs C : [eine kompakte Einführung in die Programmiersprache C ; vom ersten Schritt bis zum komplexen Programm ; mit Übungen zur Lernkontrolle, Level 1 - 3]
Sehr preiswerte Einführung in die Grundelemente der prozeduralen Programmiersprache C. System- und Hardwareprogrammierung sowie die Programmierung grafischer Oberflächen werden nicht behandelt. (Klaus Barckow)
C ist eine 1972 entwickelte, standardisierte Programmiersprache, die auf fast allen Computersystemen verfügbar ist. Viele Sprachen, z.B. C++ und C [sharp], aber auch Java orientieren sich an C. Ein umfassendes Handbuch zu C ist H. Herold (BA 10/10), von dem auch eine gute C-Referenz stammt (ID 32/05). Das vorliegende sehr preiswerte Taschenbuch will Einsteiger ohne besondere Vorkenntnisse in die Grundlagen von C einführen. Auf Details wie die System- und Hardwareprogrammierung sowie die Programmierung grafischer Benutzeroberflächen wird bewusst verzichtet. Die Ausführungen basieren auf dem aktuellen Standard C99. Didaktisch sehr gut aufgebaut. Zu jedem Kapitel gibt es Aufgaben von unterschiedlichem Schwierigkeitsgrad. Aufgaben mit Level 1 widmen sich einfachen Grundfragen und dienen der Verständniskontrolle. Im Level 2 werden Codeabschnitte gezeigt mit der Aufforderung, falsch oder sehr schlecht programmierte Abschnitte herauszufinden und zu berichtigen oder zu verbessern. Im Praxislevel 3 gilt es dann, anhand einer Aufgabenstellung ein eigenes Listing zu entwerfen. Auch zum Selbststudium einsetzbar.(2) (Klaus Barckow)
