Windows Vista : [das Buch für den Systemadministrator ; die neuen Funktionalitäten ; Core Features für den Systemadministrator ; auf CD: Vollversionen der Lernvideos zu den WDS (Windows-Bereitstellungsdiensten) und IPv6]
Der Autor wendet sich an Systemadministratoren, beginnt aber mit einer Übersicht aller neuen Features von Windows Vista und die verschiedenen Installationsmöglichkeiten, Informationen, die vor allem Einzelplatznutzer interessieren und Büchern wie Ph. Kiefer (BA 4/07) oder W. Gieseke (BA 4/07) entnommen werden können. Im 2. Teil des Buches liegt der Schwerpunkt dann aber detailliert bei den Features, die für Aufbau, Verwaltung und Wartung eines Netzwerkes von Bedeutung sind, z.B. den neuen Windows Bereitstellungsdiensten (WDS) zur Installation von Netzwerken sowie bei Sicherheitsfunktionen. Gut herausgearbeitet werden die Überwachung der Systemleistung, Remoteverbindungen, Netzwerktroubleshooting, Wireless-Netzwerke unter Vista, Benutzerkontensteuerung sowie die Themenkreise Netzwerk- und Freigabecenter und Quality of Service. Auch das neue Internet-Protokoll IPv6 wird ausführlich behandelt. Gut nachvollziehbare Darstellung, die keine besonderen Kenntnisse voraussetzt und sich auf viele Bildschirmabbildungen stützt. Auf CD-ROM Lernvideos über das Protokoll IPv6 und die Bereitstellungsdienste WDS. (2) (Klaus Barckow)
