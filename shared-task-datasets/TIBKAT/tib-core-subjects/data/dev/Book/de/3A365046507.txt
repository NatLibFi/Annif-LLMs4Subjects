Windows Server 2003 : [der methodische und ausführliche Einstieg]
Seit Mai 2003 gibt es in Deutschland die nächste Generation von Microsofts Server-Betriebssystem mit dem Namen Windows Server 2003. Dieser in 4 Ausführungen angebotene Nachfolger von Windows 2000 Server soll die Sicherheit verbessern und für eine höhere Stabilität sorgen. Auf der Grundlage der Standard Edition wird den Lesern, nach einer kurzen Einführung in das System und die Netzwerktechnik, das Herzstück eines Windows-Servers, der Verzeichnis-Dienst Active Directory Services (ADS), erläutert. Planung, Konfiguration und Verwaltung von ADS werden ausführlich dargestellt. Die Organisation der Zugriffsrechte, Verwaltung der Druckausgabe, Profile und Richtlinien sowie die Systemverwaltung folgen im Anschluss. Im Unterschied zur vorhergehenden Ausgabe (Bernd Zoller: BA 11/00) müssen Neueinsteiger wesentlich mehr an Vorwissen mitbringen. Das Buch kommt Umsteigern von Windows NT und Anwendern mit Vorerfahrung beim Einsatz von Windows 2000 Server entgegen. Ein Austausch des Vorgängertitels ist noch nicht notwendig, aber es empfiehlt sich den Bestand durch die neue Ausgabe zu aktualisieren. (2)
