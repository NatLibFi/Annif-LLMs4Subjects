SQL Server Administration : Insider-Wissen - praxisnah & kompetent
SQL Server ist ein relationalen Datenbankmanagementsystem von Microsoft, das in seiner aktuellen Version 2017 in mehreren Editionen mit unterschiedlichem Leistungsumfang vorliegt. Das Buch wendet sich an Leser mit umfangreichen Vorkenntnissen in der Datentechnik und in früheren Versionen von SQL Server und ist somit, namentlich in gröe︢ren Bibliotheken, eine gute Ergänzung zu dem Einführungswerk für Einsteiger in SQL Server 2017 von K. Konopasek und zu R. Panther - zwar nur Version 2016, aber mit sehr vielen Beispielen und Aufgaben mit Lösungen). Bücher zu früheren Versionen wie etwa U. Boddenberg sollten ersetzt werden. Ausführlich erläutert werden die zahlreichen in der aktuellen Version hinzugekommenen Funktionen mit ihren Einsatzmöglichkeiten. Schwerpunkte liegen beim Azure-Clouddienst, der Fehlerbehebung von Anwendungen, beim Schutz des Servers und seiner Daten, bei der Leistungsoptimierung des SQL-Servers, bei Hochverfügbarkeit und bei der über die normale Sicherung hinausgehende Notfallwiederherstellung, jeweils getrennt nach den verschiedenen Editionen
